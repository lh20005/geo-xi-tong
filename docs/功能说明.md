# GEO优化系统 - 功能详细说明

## 一、系统概述

GEO优化系统是一个完整的AI驱动的内容优化工具，从关键词分析到文章生成，提供端到端的解决方案。系统集成了企业图库和知识库管理，帮助企业更好地组织和利用资源。

## 二、核心功能模块

### 1. 工作台 (Dashboard)
- 系统概览和快速导航
- 功能统计卡片展示
- 快速开始引导
- 系统特点介绍

### 2. API配置 (ConfigPage)
- 支持DeepSeek、Gemini、Ollama三种AI模型
- API密钥安全管理
- 本地模型自动检测
- 连接测试功能

- 多模型支持
- 使用统计和推荐
- 筛选和搜索功能

### 4. 话题管理 (TopicsPage)
- 可视化话题列表
- 编辑和删除话题
- 批量选择功能

### 5. 文章生成 (ArticlePage)
- AI驱动的内容创作
- 自定义生成要求
- 知识库智能引用
- 一键复制下载
- 支持配图

### 6. 文章管理 (ArticleListPage)
- 文章列表展示
- 详情查看
- 历史记录管理
- 批量操作
- 按任务筛选

### 7. 文章生成任务 (TaskManagementPage)
- 批量文章生成
- 任务状态监控
- 智能选择蒸馏结果
- 任务诊断和重试
- 批量删除管理

### 8. 文章设置 (ArticleSettingsPage)
- 文章模板管理
- 字数和风格配置
- 自定义要求设置
- 模板复用

### 9. 转化目标 (ConversionTargetsPage)
- 转化目标管理

## 三、文章生成任务详解

### 3.1 核心价值
- **批量生成**: 一次性生成多篇文章，提高效率
- **智能选择**: 自动选择合适的蒸馏结果
- **任务监控**: 实时查看生成进度和状态
- **错误处理**: 自动诊断问题，支持重试

### 3.2 任务创建流程
```
1. 选择蒸馏历史（关键词来源）
   ↓
2. 选择图库（配图来源）
   ↓
3. 选择知识库（内容参考）
   ↓
4. 选择文章设置（模板）
   ↓
5. 选择转化目标（可选）
   ↓
6. 设置文章数量
   ↓
7. 创建任务，后台自动生成
```

### 3.3 任务状态说明
| 状态 | 说明 | 操作 |
|------|------|------|
| pending | 等待执行 | 可删除 |
| running | 正在执行 | 不可操作 |
| completed | 已完成 | 可查看、删除、重试 |
| failed | 执行失败 | 可诊断、重试、删除 |

### 3.4 任务管理功能
- **查看详情**: 查看任务配置和生成的文章
- **诊断任务**: 自动检测问题并提供解决建议
- **重试任务**: 重新执行失败或已完成的任务
- **删除任务**: 单个删除或批量删除
- **筛选查看**: 按任务查看生成的文章

## 四、文章设置详解

### 4.1 功能特点
- **模板管理**: 创建和管理文章生成模板

- **灵活调整**: 随时修改模板配置

### 4.2 设置字段说明

## 五、转化目标详解

### 5.1 核心价值
- **转化嵌入**: 在文章中自然嵌入转化信息
- **多种类型**: 支持电话、邮箱、网址、微信等

### 5.4 最佳实践

## 十、常见问题

### Q1: 如何创建文章生成任务？
**A**: 
1. 进入"文章生成任务"页面
2. 点击"创建任务"
3. 依次选择蒸馏历史、图库、知识库、文章设置
4. 可选择转化目标
5. 设置文章数量
6. 点击"创建"，系统自动生成

### Q2: 任务失败了怎么办？
**A**: 
1. 点击"诊断"按钮查看问题
2. 根据诊断建议修复问题
3. 点击"重试"重新执行任务
4. 如果多次失败，检查AI配置和网络连接

### Q3: 如何批量删除任务？
**A**: 
1. 在任务列表中勾选要删除的任务
2. 点击"批量删除"按钮
3. 确认删除操作
4. 注意：正在运行的任务无法删除

### Q4: 知识库支持哪些文件格式？
**A**: 支持.txt、.md、.pdf、.doc、.docx五种格式，单个文件最大10MB。

### Q5: 如何确保AI生成的文章准确？
**A**: 
1. 上传准确的企业知识文档
2. 选择相关的知识库
3. 提供详细的文章要求
4. 人工审核和优化

### Q6: 知识库的文档会被如何使用？
**A**: 文档内容会在AI生成文章时作为参考资料，确保生成的内容符合企业实际情况和专业标准。

### Q7: 可以上传多少个文档？
**A**: 没有数量限制，但建议保持知识库精简，只上传高质量、相关的文档。

### Q8: 图库的图片可以在哪里使用？
**A**: 图库用于文章配图。在创建生成任务时选择图库，系统会自动为生成的文章配图。

### Q9: 转化目标如何在文章中体现？
**A**: 在创建生成任务时选择转化目标，AI会自然地将转化信息（如联系方式）融入文章内容中。

### Q10: 一个任务最多可以生成多少篇文章？
**A**: 单个任务最多可以生成100篇文章。建议根据实际需求合理设置数量，通常10-50篇为宜。
