# GEO优化系统 - UI设计说明

## 一、设计理念

### 1.1 核心原则
- **专业性**: 体现企业级工具的专业感
- **高端感**: 使用渐变色和现代化设计元素
- **易用性**: 清晰的信息层级和操作流程
- **一致性**: 统一的设计语言和交互模式

### 1.2 设计关键词
专业 | 智能 | 高效 | 现代 | 简洁

## 五、交互设计

### 5.1 动画效果

#### 页面过渡
```css
transition: all 0.3s ease;
```

#### 按钮悬停
```css
transition: all 0.2s ease;
transform: translateY(-2px); /* 轻微上移 */
```

#### 卡片悬停
```css
transition: all 0.3s ease;
transform: translateY(-4px);
box-shadow: 0 8px 16px rgba(0,0,0,0.15);
```

#### 列表项选中
```css
transition: all 0.2s ease;
border-color: #0ea5e9;
background: rgba(14, 165, 233, 0.05);
```

### 5.2 反馈机制

#### 成功操作
- Toast提示: 绿色，右上角
- 图标: ✓ CheckCircle
- 持续时间: 3秒

#### 错误操作
- Toast提示: 红色，右上角
- 图标: ✗ CloseCircle
- 持续时间: 5秒

#### 警告提示
- Toast提示: 黄色，右上角
- 图标: ⚠ Warning
- 持续时间: 4秒

#### 确认操作
- Modal对话框
- 标题: "确认{操作}"
- 内容: 操作说明
- 按钮: 取消 + 确认

### 5.3 加载状态

#### 按钮加载
```
[⟳ 加载中...]
禁用状态，显示旋转图标
```

#### 页面加载
```
居中显示Spin
下方提示文字
```

#### 列表加载
```
Skeleton占位符
模拟实际内容结构
```

## 六、响应式设计

### 6.1 断点设置

```
xs: < 576px   (手机)
sm: ≥ 576px   (大手机)
md: ≥ 768px   (平板)
lg: ≥ 992px   (桌面)
xl: ≥ 1200px  (大桌面)
xxl: ≥ 1600px (超大桌面)
```

### 6.2 布局适配

#### 桌面 (≥ 992px)
- 侧边栏: 固定240px
- 内容区: 自适应
- 卡片: 多列布局

#### 平板 (768px - 991px)
- 侧边栏: 可折叠
- 内容区: 自适应
- 卡片: 2列布局

#### 手机 (< 768px)
- 侧边栏: 抽屉式
- 内容区: 单列
- 卡片: 单列布局

## 七、可访问性

### 7.1 颜色对比度
- 文字与背景对比度 ≥ 4.5:1
- 大文字对比度 ≥ 3:1

### 7.2 键盘导航
- 所有交互元素可通过Tab访问
- 焦点状态清晰可见
- 支持Enter/Space触发

### 7.3 语义化
- 使用语义化HTML标签
- 提供alt文本
- ARIA标签支持
