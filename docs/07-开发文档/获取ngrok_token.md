# 获取 ngrok authtoken

## 步骤

### 1. 访问 ngrok 网站

打开浏览器，访问：https://ngrok.com

### 2. 注册账号

- 点击右上角 "Sign up" 按钮
- 可以使用 Google 账号快速注册
- 或者使用邮箱注册

### 3. 登录后获取 authtoken

注册并登录后，会自动跳转到 Dashboard 页面，或者：

1. 点击左侧菜单 "Your Authtoken"
2. 或访问：https://dashboard.ngrok.com/get-started/your-authtoken

### 4. 复制 authtoken

你会看到类似这样的内容：

```
Your Authtoken

2abc123def456ghi789jkl012mno345pqr678stu901vwx234yz
```
