# 蒸馏显示问题 - 用户操作指南

**问题**: 蒸馏管理页面显示"暂无蒸馏结果"  
**原因**: 前端缓存问题  
**解决**: 清除缓存后重启应用

---

## 快速修复（3 步）

### 步骤 1: 完全退出应用

1. 点击应用窗口右上角的 **×** 按钮
2. 确保应用图标从任务栏/Dock 消失
3. 等待 3 秒

### 步骤 2: 清除缓存

**macOS 用户**:
```bash
rm -rf ~/Library/Application\ Support/ai-geo-system/Cache
rm -rf ~/Library/Application\ Support/ai-geo-system/Code\ Cache
```

**或使用自动脚本**:
```bash
cd windows-login-manager
./fix-distillation-cache.sh
```

**Windows 用户**:
1. 按 `Win+R` 打开运行对话框
2. 输入 `%APPDATA%\ai-geo-system` 并回车
3. 删除 `Cache` 和 `Code Cache` 文件夹

### 步骤 3: 重新启动应用

1. 重新打开应用
2. 登录账号（如果需要）
3. 进入"蒸馏管理"页面
4. 应该能看到所有蒸馏结果了！

## 如果仍然看不到

### 方法 1: 使用开发者工具

1. 在应用中按 `Cmd+Option+I` (macOS) 或 `Ctrl+Shift+I` (Windows)
2. 在 Console 标签中输入并执行：
   ```javascript
   localStorage.clear();
   sessionStorage.clear();
   location.reload();
   ```

### 方法 2: 强制刷新

1. 进入蒸馏管理页面
2. 按住 `Shift` 键
3. 点击右上角的"刷新"按钮
