## 技术说明

### 问题原因

数据库中的 `topic_count` 字段没有自动更新，导致 API 查询时过滤掉了有话题的蒸馏记录。

### 修复措施

1. **数据修复**: 更新了所有用户的 `topic_count` 字段
2. **触发器**: 创建了自动更新触发器，以后插入或删除话题时会自动更新计数
3. **迁移文件**: 更新了数据库迁移文件，确保新环境不会出现同样问题
