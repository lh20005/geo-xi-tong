# 蒸馏结果模块使用指南

## 概述

"蒸馏结果"是一个独立的功能模块，用于展示、管理和操作所有历史蒸馏记录。该模块从"关键词蒸馏"页面中分离出来，提供更专注的结果管理体验。

## 功能特性

### 1. 历史记录列表
- 显示所有蒸馏记录（关键词、话题数量、AI模型、创建时间）
- 支持分页浏览（每页10条）
- 自动按创建时间降序排列

### 2. 记录详情查看
- 点击"查看详情"查看完整话题列表
- 显示所有AI生成的问题
- 支持滚动浏览长列表

### 3. 记录管理
- **编辑关键词**: 修改记录的关键词描述
- **删除单条**: 删除指定的蒸馏记录
- **批量删除**: 一键删除所有历史记录

### 4. 快速跳转
- **查看话题**: 跳转到话题详情页面
- **生成文章**: 跳转到文章生成页面

## 使用流程

### 步骤1: 执行蒸馏
1. 访问侧边栏"关键词蒸馏"页面
2. 输入您的关键词（如：英国留学、Python培训）
3. 点击"开始蒸馏"按钮
4. 等待AI生成话题

### 步骤2: 查看结果
1. 蒸馏成功后，点击提示消息中的"查看结果"链接
2. 或者点击侧边栏的"蒸馏结果"菜单项
3. 在历史记录列表中找到您的记录
4. 点击"查看详情"查看完整话题列表

### 步骤3: 管理记录
- **编辑关键词**: 点击记录行的"编辑"按钮，输入新关键词并保存
- **删除记录**: 点击"删除"按钮，确认后删除该记录
- **删除所有**: 点击页面右上角的"全部删除"按钮

### 步骤4: 后续操作
- 点击详情卡片中的"查看话题"按钮，进入话题详情页
- 点击"生成文章"按钮，基于话题生成文章

## 界面说明

### 历史记录表格
| 列名 | 说明 |
|------|------|
| 关键词 | 蒸馏时输入的关键词 |
| 话题数量 | 生成的话题数量 |
| AI模型 | 使用的AI提供商（DeepSeek/Gemini/Ollama） |
| 创建时间 | 记录创建的时间 |
| 操作 | 查看详情、编辑、删除按钮 |

### 详情卡片
- **标题**: 显示关键词
- **话题列表**: 所有生成的问题，每个问题独立显示
- **操作按钮**: 
  - 查看话题：跳转到话题页面
  - 生成文章：跳转到文章生成页面
  - 编辑关键词：修改关键词
  - 删除记录：删除该记录

## 数据持久化

系统会自动保存您当前查看的记录：
- 刷新页面后，上次查看的记录会自动恢复
- 关闭浏览器后重新打开，状态依然保留
- 删除记录后，保存的状态会自动清除

## 注意事项

1. **删除操作不可恢复**: 删除记录后无法恢复，请谨慎操作
2. **关键词不能为空**: 编辑关键词时，必须输入有效内容
3. **数据同步**: 编辑或删除操作会立即同步到数据库
4. **分页浏览**: 记录较多时，使用表格底部的分页控件浏览

## 常见问题

**Q: 为什么看不到我刚才的蒸馏结果？**
A: 请点击侧边栏的"蒸馏结果"菜单，或点击蒸馏成功提示中的"查看结果"链接。

**Q: 如何找到之前的蒸馏记录？**
A: 所有历史记录都在"蒸馏结果"页面的表格中，按创建时间降序排列。

**Q: 删除记录后能恢复吗？**
A: 不能。删除操作是永久性的，请在删除前确认。

**Q: 编辑关键词会影响已生成的话题吗？**
A: 不会。编辑关键词只修改显示的标签，不会改变已生成的话题内容。
