### 任务状态 → 可用操作

| 状态 | 图标 | 可用操作 |

## ⚡ 常用操作流程

### 🛑 终止卡住的任务
```
1. 找到"执行中"的任务
2. 点击红色"终止"按钮
3. 确认 → 完成
```

### 🗑️ 清理历史任务
```
方式1: 批量删除
  勾选任务 → 批量删除 → 确认

方式2: 一键清空
  删除全部 → 确认 → 完成
```

### 📊 查看任务详情
```
点击"日志" → 查看执行记录 → 关闭
```

|------|------|----------|

| ⚪ 灰色 | 不可用 | - |

## ⚠️ 安全提示

### ✅ 安全操作
- 查看日志
- 刷新列表
- 勾选任务

### ⚠️ 需确认
- 执行任务
- 取消任务

### 🚨 危险操作
- 终止任务（不可恢复）
- 删除任务（不可恢复）
- 批量删除（不可恢复）
- 删除全部（不可恢复）

## 💡 快捷技巧

### 键盘操作
- `Ctrl/Cmd + A` - 全选（在表格内）
- `Esc` - 关闭弹窗

### 鼠标操作
- 单击复选框 - 选择任务
- 双击任务行 - 查看详情（未来功能）

### 批量选择
- 表头复选框 - 全选/取消全选
- 单个复选框 - 逐个选择

## 🎯 使用建议

### 日常使用
1. 定期查看任务状态
2. 及时处理失败任务
3. 定期清理历史记录

### 问题处理
1. 任务卡住 → 终止 → 重新创建
2. 任务失败 → 查看日志 → 修复问题
3. 列表太多 → 批量删除 → 保持整洁

### 最佳实践
- ✅ 执行前检查账号状态
- ✅ 定期查看任务日志
- ✅ 及时清理无用任务
- ❌ 不要频繁终止任务
- ❌ 不要删除未完成的任务
