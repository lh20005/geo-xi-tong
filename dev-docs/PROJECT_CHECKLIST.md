## 🚀 快速开始

### 1. 安装依赖
```bash
npm run install:all
```

### 2. 配置环境
```bash
cp .env.example .env
# 编辑 .env 文件
```

### 3. 创建数据库
```bash
createdb geo_system
```

### 4. 运行迁移
```bash
cd server && npm run db:migrate
```

### 5. 启动服务
```bash
npm run dev
```

### 6. 访问系统
- 前端: http://localhost:5173
- 后端: http://localhost:3000
