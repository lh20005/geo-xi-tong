# ⚠️ 重要：Ollama配置保存修复

## 🚨 如果你遇到以下问题

- ✅ Ollama连接测试成功
- ❌ 点击"保存配置"后提示失败
- ❌ 无法使用Ollama进行AI任务

## ✅ 快速修复（1分钟）

打开终端，运行以下命令：

```bash
cd server
npm run db:fix:constraint
```

然后重启服务器：

```bash
npm run dev
```

## 🎯 这会做什么？

修复脚本会：
1. ✅ 更新数据库约束定义
2. ✅ 清理不一致的数据
3. ✅ 确保Ollama配置可以正常保存

## 📋 验证修复

修复后，测试以下操作：

1. **保存配置**
   - 访问配置页面
   - 选择"本地Ollama"
   - 选择模型
   - 点击"保存配置"
   - ✅ 应该显示"API配置保存成功！"

2. **使用功能**
   - 进入"关键词蒸馏"
   - 输入测试关键词
   - ✅ 应该成功生成问题列表

## 🤔 为什么需要修复？

数据库约束定义有一个小问题，导致保存Ollama配置时失败。这个修复脚本会更新约束定义，让一切正常工作。
