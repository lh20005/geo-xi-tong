## 📝 下一步操作

### 1. 配置AI API（必需）

在使用系统前，您需要配置AI API密钥：

1. 打开浏览器访问 http://localhost:5173
2. 点击左侧菜单"API配置"
3. 选择AI模型（DeepSeek或Gemini）
4. 输入您的API密钥
5. 点击"保存配置"

**获取API密钥**：
- **DeepSeek**: https://platform.deepseek.com
- **Gemini**: https://makersuite.google.com/app/apikey

### 2. 开始使用

配置完API后，您可以：

1. **关键词蒸馏**
   - 点击"关键词蒸馏"
   - 输入关键词，如"英国留学"
   - 点击"开始蒸馏"
   - 查看生成的话题

2. **管理话题**
   - 查看蒸馏结果
   - 编辑或删除话题
   - 选择话题用于文章生成

3. **生成文章**
   - 选择话题后点击"生成文章"
   - 输入文章要求
   - 等待AI生成
   - 复制或下载文章
