## 常见问题排查

### 问题1：点击导航没反应
**可能原因：**
- JavaScript 未加载
- 浏览器控制台有错误

**解决方法：**
- 打开浏览器开发者工具（F12）
- 查看 Console 是否有错误
- 刷新页面重试

### 问题2：跳转到首页但没有滚动
**可能原因：**
- 锚点 ID 不存在
- 延迟时间不够

**解决方法：**
- 检查首页是否有对应的 ID（features、advantages、pricing）
- 增加延迟时间（目前是 100ms）

### 问题3：在首页点击导航会刷新页面
**可能原因：**
- 使用了 `<a>` 标签而不是 `<button>`
- 没有正确判断当前路径

**解决方法：**
- 确认 Header 组件使用了 `handleNavigateToSection` 函数
- 确认有判断 `window.location.pathname === '/'`
