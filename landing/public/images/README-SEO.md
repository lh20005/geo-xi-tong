# SEO 图片资源说明

## 需要添加的图片

### og-image.png (必需)
- 尺寸：1200 x 630 像素
- 用途：社交媒体分享时显示的预览图
- 建议内容：品牌 Logo + 产品名称 + 核心价值主张
- 格式：PNG 或 JPG

### 示例设计建议
- 背景：渐变蓝紫色（与品牌一致）
- 主标题：GEO优化系统
- 副标题：AI时代品牌推荐优化工具
- Logo：放置在左上角或中央

## 部署前检查清单

- [x] 域名已配置为 `jzgeo.cc`
- [ ] 添加 og-image.png 图片到 `landing/public/images/` 目录
- [ ] 确保 HTTPS 证书已配置
