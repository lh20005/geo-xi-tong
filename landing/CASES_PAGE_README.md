#### 配色方案
- 蓝色系：AI推荐率
- 绿色系：咨询/转化量
- 紫色系：成本/客单价

### 4. 图片来源

所有图片来自 **Unsplash** (免费商用图库)：

```
https://images.unsplash.com/photo-{id}?w=800&q=80
```

**优点**:
- ✅ 完全免费
- ✅ 无需署名
- ✅ 可商用
- ✅ 高质量专业图片
- ✅ 自动优化（通过URL参数）
