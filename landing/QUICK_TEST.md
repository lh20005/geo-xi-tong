## 🐛 如果遇到问题

### 问题1：点击导航没反应
**解决方法：**
1. 打开浏览器开发者工具（F12）
2. 查看 Console 是否有错误
3. 刷新页面重试

### 问题2：跳转但不滚动
**检查：**
1. 首页是否有对应的锚点 ID（features、advantages、pricing）
2. 浏览器控制台是否有错误

### 问题3：服务启动失败
**解决方法：**
```bash
# 重新安装依赖
cd landing
rm -rf node_modules
npm install
npm run dev
```
