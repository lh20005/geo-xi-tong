# 模板按钮使用指南

## 快速开始

### 1. 打开新建对话框

进入"文章设置"页面 → 点击"新建文章设置"按钮

### 2. 看到三个模板按钮

在提示词输入框下方，你会看到：

```
快速选择模板（点击按钮自动填充）：

[⚖️ 平衡型]  [🎓 重专业]  [📢 重营销]
```

### 3. 点击任意按钮

- 点击 **⚖️ 平衡型** → 自动填充平衡型模板
- 点击 **🎓 重专业** → 自动填充重专业模板
- 点击 **📢 重营销** → 自动填充重营销模板

### 4. 查看效果

提示词输入框会自动填充完整的模板内容

### 5. 保存或修改

- **直接保存**：如果满意，输入设置名称，点击"创建"
- **修改后保存**：可以在模板基础上调整，然后保存

## 三个模板的区别

### ⚖️ 平衡型（推荐日常使用）

- **公司名称**：出现1-2次
- **风格**：专业 + 适度营销
- **字数**：1500-2500字
- **适合**：日常内容生成

### 🎓 重专业（推荐专业文章）

- **公司名称**：不出现
- **风格**：纯专业，深度分析
- **字数**：2000-3000字
- **适合**：行业分析、技术文章

### 📢 重营销（推荐产品推广）

- **公司名称**：出现3-4次
- **风格**：营销推广为主
- **字数**：1500-2000字
- **适合**：产品推广、品牌宣传

## 常见问题

### Q1: 点击按钮后可以修改吗？

**答**：可以！模板只是提供一个起点，你可以随意修改。

### Q2: 可以切换模板吗？

**答**：可以！点击另一个按钮，会替换当前内容。

### Q3: 模板会丢失吗？

**答**：不会！模板是固定的，永远可以一键调用。

### Q4: 必须使用模板吗？

**答**：不必须！你也可以完全自己编写提示词。

### Q5: 保存后还能用模板吗？

**答**：可以！编辑文章设置时，仍然可以点击模板按钮。

## 使用技巧

### 技巧1：从模板开始

即使你想自定义，也建议先点击一个最接近的模板，然后修改。这样更高效。

### 技巧2：混合使用

可以参考多个模板的内容，创建自己的混合版本。

### 技巧3：创建多个设置

为不同场景创建不同的文章设置：
- "日常文章" - 使用平衡型
- "专业分析" - 使用重专业
- "产品推广" - 使用重营销

### 技巧4：保存常用版本

如果你经常使用某个修改后的版本，保存为独立的文章设置，方便重复使用。

## 示例流程

### 示例1：创建日常文章设置

1. 点击"新建文章设置"
2. 点击"⚖️ 平衡型"按钮
3. 输入名称："日常文章-平衡型"
4. 点击"创建"
5. 完成！

### 示例2：创建自定义设置

1. 点击"新建文章设置"
2. 点击"⚖️ 平衡型"按钮（作为起点）
3. 修改提示词，调整字数要求为"1000-1500字"
4. 修改公司名称要求为"只出现1次"
5. 输入名称："短文章-轻营销"
6. 点击"创建"
7. 完成！

## 界面说明

### 按钮颜色

- **蓝色**（⚖️ 平衡型）：中性，适合大多数场景
- **绿色**（🎓 重专业）：专业，纯内容
- **橙色**（📢 重营销）：活跃，营销推广

### 鼠标悬停

将鼠标悬停在按钮上，会显示详细说明：
- 平衡型：专业内容与适度营销相结合，公司名称出现1-2次
- 重专业：纯专业内容，深度分析，不做营销推广
- 重营销：以营销推广为主，公司名称出现3-4次

### 占位符提示

在标签旁边会显示支持的占位符：
```
（支持占位符：{keyword}, {topics}, {companyName}, {knowledgeBase}）
```
