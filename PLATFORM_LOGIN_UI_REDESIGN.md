# 平台登录页面 UI 重新设计

## 设计目标
重新设计平台登录页面的UI布局，使其与系统其他页面保持统一协调的视觉风格。

## 主要改进

### 1. 顶部统计卡片
- **新增**：4个统计卡片展示关键指标
  - 支持平台（蓝色 #0ea5e9）
  - 已配置平台（绿色 #52c41a）
  - 账号总数（蓝色 #1890ff）
  - 活跃账号（橙色 #faad14）
- **参考**：与文章列表页的统计卡片风格一致

### 2. 平台登录卡片区域
- **优化**：使用Card组件包裹，添加标题和刷新按钮
- **图标**：使用 LoginOutlined 图标，颜色 #0ea5e9
- **提示文字**：添加操作说明，颜色 #64748b
- **卡片样式**：
  - 已登录：绿色边框 + 浅绿背景 (#f6ffed)
  - 未登录：灰色边框 (#e2e8f0) + 白色背景
  - 圆角统一为 8px
  - 图标大小调整为 64x64

### 3. 账号管理表格
- **标题**：添加 StarFilled 图标，颜色 #0ea5e9
- **表格列**：
  - 所有列居中对齐
  - 平台名称使用蓝色Tag显示
  - 状态使用 success/default Tag
  - 时间格式统一为中文本地化
  - 未使用时显示灰色"未使用"文字
- **分页**：添加快速跳转和每页条数选择

### 4. 颜色方案统一
- 主色调：#0ea5e9（蓝色）
- 成功色：#52c41a（绿色）
- 警告色：#faad14（橙色）
- 文字色：#1e293b（深色）、#64748b（灰色）、#94a3b8（浅灰）
- 边框色：#e2e8f0

### 5. 布局间距
- 页面内边距：24px
- 卡片间距：16px（gutter）
- 卡片底部间距：24px

## 设计参考
- 文章列表页（ArticleListPage.tsx）：统计卡片、筛选区域
- 蒸馏页面（DistillationPage.tsx）：卡片标题样式、表格分页
- 话题页面（TopicsPage.tsx）：卡片布局、操作按钮

## 视觉效果
1. **统一性**：与其他页面保持一致的视觉语言
2. **层次感**：通过颜色和间距区分不同功能区域
3. **可读性**：合理的字体大小和颜色对比
4. **交互性**：清晰的状态反馈和操作提示

## 技术实现
- 使用 Ant Design 组件库
- 响应式布局（xs/sm/md/lg）
- TypeScript 类型安全
- 统一的样式常量
