# 转化目标频率优化说明

## 问题描述

在v2.0版本中，AI被要求"必须执行"融入公司名称，导致AI过度遵循指令，在文章的每个段落都提及公司名称，造成过度营销的感觉，影响文章的专业性和可读性。

## 优化方案

### 修改前（v2.0）

```typescript
prompt += `5. 【必须执行】在文章中自然地融入公司名称"${companyName}"，起到宣传推广作用\n`;
prompt += `   - 可以在介绍相关服务、产品或解决方案时提及\n`;
prompt += `   - 要确保提及方式自然、不生硬\n`;
prompt += `   - 至少出现1-2次\n`;

// ...

prompt += `\n\n【转化目标】\n`;
prompt += `本文的宣传目标是：${companyName}\n`;
prompt += `请在文章中自然地提及该公司，展示其专业性和优势。`;
```

**问题**：
- 使用"必须执行"这样的强制性词汇
- 两次强调转化目标
- AI理解为"尽可能多地提及"
- 结果：每段都出现公司名称

### 修改后（v2.1）

```typescript
prompt += `5. 【重要】在文章中适度提及公司名称"${companyName}"，起到宣传推广作用\n`;
prompt += `   - 只在1-2个合适的地方提及，不要每段都出现\n`;
prompt += `   - 可以在介绍解决方案或给出建议时自然提及\n`;
prompt += `   - 提及方式要自然、不生硬、不重复\n`;
prompt += `   - 避免过度营销，保持文章的专业性和可读性\n`;

// ...

prompt += `\n\n【转化目标提示】\n`;
prompt += `本文需要适度提及：${companyName}\n`;
prompt += `注意：只在1-2个自然的位置提及即可，不要过度重复。重点是文章质量，而不是频繁提及公司名称。`;
```

**改进**：
- 将"必须执行"改为"重要"，降低强制性
- 明确说明"只在1-2个合适的地方提及"
- 强调"不要每段都出现"
- 增加"避免过度营销"的要求
- 强调"重点是文章质量"

## 关键优化点

### 1. 降低强制性

| 修改前 | 修改后 |
|--------|--------|
| 【必须执行】 | 【重要】 |
| 至少出现1-2次 | 只在1-2个合适的地方提及 |

### 2. 明确限制频率

**新增的明确指令**：
- "只在1-2个合适的地方提及"
- "不要每段都出现"
- "不要过度重复"

### 3. 强调质量优先

**新增的质量要求**：
- "避免过度营销"
- "保持文章的专业性和可读性"
- "重点是文章质量，而不是频繁提及公司名称"

### 4. 调整提示词结构

| 修改前 | 修改后 |
|--------|--------|
| 【转化目标】 | 【转化目标提示】 |
| 请在文章中自然地提及该公司 | 只在1-2个自然的位置提及即可 |
| 展示其专业性和优势 | 不要过度重复 |

## 预期效果

### 文章示例对比

#### v2.0版本（过度营销）

```
标题：如何选择靠谱的英国留学机构

第一段：随着英国留学热度的持续升温，优学留学咨询发现越来越多的学生...

第二段：在众多留学机构中，优学留学咨询凭借其专业的团队...

第三段：优学留学咨询的顾问团队认为，选择留学机构时需要...

第四段：综合来看，优学留学咨询在这些方面都表现出色...
```

❌ 问题：每段都出现公司名称，过度营销

#### v2.1版本（适度营销）

```
标题：如何选择靠谱的英国留学机构

第一段：随着英国留学热度的持续升温，越来越多的学生和家长开始关注...

第二段：在众多留学机构中，优学留学咨询凭借其专业的团队和丰富的经验...

第三段：选择留学机构时，需要重点关注以下几个方面...

第四段：综合来看，专业的留学服务能够大大提高申请成功率...
```

✅ 改进：只在第二段提及公司名称，自然且不突兀

## 测试验证

### 验证清单

生成文章后，检查以下几点：

- [ ] 公司名称出现次数：1-2次 ✅
- [ ] 不是每段都出现 ✅
- [ ] 提及方式自然 ✅
- [ ] 不影响文章可读性 ✅
- [ ] 保持专业性 ✅
- [ ] 没有过度营销感 ✅

### 如果仍然出现过多

如果AI仍然在每段都提及公司名称，可以进一步优化：

```typescript
// 更强的限制
prompt += `5. 【注意】在文章中最多提及1-2次公司名称"${companyName}"\n`;
prompt += `   - 严格限制：整篇文章中公司名称最多出现2次\n`;
prompt += `   - 大部分段落不应该出现公司名称\n`;
prompt += `   - 只在最合适的1-2个位置自然提及\n`;
```

## 总结

通过这次优化，我们：

1. **降低了强制性**：从"必须执行"改为"重要"
2. **明确了频率限制**：只在1-2个位置提及
3. **强调了质量优先**：避免过度营销
4. **改进了提示词结构**：更清晰的指令

这样生成的文章既能起到宣传推广作用，又能保持专业性和可读性，达到更好的营销效果。
