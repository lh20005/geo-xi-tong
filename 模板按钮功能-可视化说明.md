## 🎯 按钮详细说明

### 1. ⚖️ 平衡型按钮
```
┌──────────────────────────────────────┐
│  ⚡ ⚖️ 平衡型                        │
│  ────────────────────────────────    │
│  专业内容与适度营销相结合，          │
│  公司名称出现1-2次                   │
└──────────────────────────────────────┘
```
- **颜色**: 蓝色 (#1890ff)
- **图标**: ThunderboltOutlined + ⚖️
- **用途**: 既要专业又要营销的场景
- **特点**: 平衡专业性和营销性

### 2. 🎓 重专业按钮
```
┌──────────────────────────────────────┐
│  🏆 🎓 重专业                        │
│  ────────────────────────────────    │
│  纯专业内容，深度分析，              │
│  不做营销推广                        │
└──────────────────────────────────────┘
```
- **颜色**: 绿色 (#52c41a)
- **图标**: TrophyOutlined + 🎓
- **用途**: 纯技术文章、教程、分析
- **特点**: 不提及公司名称

### 3. 📢 重营销按钮
```
┌──────────────────────────────────────┐
│  🚀 📢 重营销                        │
│  ────────────────────────────────    │
│  以营销推广为主，                    │
│  公司名称出现3-4次                   │
└──────────────────────────────────────┘
```
- **颜色**: 橙色 (#fa8c16)
- **图标**: RocketOutlined + 📢
- **用途**: 品牌推广、产品宣传
- **特点**: 多次提及公司名称

- 对话框自适应宽度


## 📐 尺寸规格

### 按钮尺寸
- 高度：32px（Ant Design 默认）
- 内边距：4px 15px
- 图标大小：14px
- 文字大小：14px

### 文本框尺寸
- 行数：16行
- 字体：monospace（等宽字体）
- 字号：13px
- 显示字符计数

### 间距
- 按钮之间间距：8px
- 标签与输入框间距：8px
- 提示文字与按钮间距：8px

## 🔍 细节优化

### 1. 文本框优化
- 使用等宽字体，便于阅读代码式的占位符
- 16行高度，可以看到完整的模板结构
- 显示字符计数，帮助用户控制长度

### 2. 按钮优化
- 使用图标增强视觉识别
- 不同颜色区分不同类型
- 提示框说明用途

### 3. 提示优化
- 在标签旁显示支持的占位符
- 在按钮上方说明"点击按钮自动填充"
- 成功提示确认操作完成

## 💡 使用建议

### 场景1：技术博客
推荐使用：🎓 **重专业**
- 纯技术内容
- 不做营销
- 深度分析

### 场景2：企业官网文章
推荐使用：⚖️ **平衡型**
- 既有专业内容
- 又有品牌露出
- 适度营销

### 场景3：产品推广文章
推荐使用：📢 **重营销**
- 突出品牌
- 引导转化
- 多次提及公司

---

**提示**: 所有模板都支持占位符，系统会在生成文章时自动替换为实际内容。
