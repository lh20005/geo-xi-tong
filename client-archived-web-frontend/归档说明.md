# Web 前端归档说明

**归档日期**: 2026-01-16  
**操作**: 将 `client/` 重命名为 `client-archived-web-frontend/`

---

## 📋 归档原因

根据架构改造方案，系统已从 Web 应用转变为桌面应用：

- ❌ **Web 前端**（此目录）- 已废弃
- ✅ **Windows 桌面客户端**（`windows-login-manager/`）- 当前使用

---

## 🎯 当前架构

### 服务器端
```
https://www.jzgeo.cc/
├── /           → 落地页（产品展示 + 用户注册）
├── /api        → 后端 API
└── /ws         → WebSocket
```

### 客户端
```
Windows 桌面应用（windows-login-manager/）
├── 完整的前端界面
├── 本地数据存储（SQLite）
├── 本地浏览器自动化
└── 本地发布执行
```

---

## ⚠️ 重要提示

1. **不要在此目录进行开发** - 所有新功能应在 `windows-login-manager/` 中实现
2. **不要部署此代码** - 服务器已移除 Web 前端支持
3. **仅作为参考和备份** - 如需查看旧代码或恢复功能时使用

---

## 🔄 如何恢复（如果需要）

```bash
# 1. 重命名回原名称
mv client-archived-web-frontend client

# 2. 安装依赖
cd client
npm install

# 3. 启动开发服务器
npm run dev
```
