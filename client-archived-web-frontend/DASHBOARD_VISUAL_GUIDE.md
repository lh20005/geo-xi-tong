  transform: translateY(-4px)
  box-shadow: 0 4px 16px rgba(0,0,0,0.12)

## 📱 响应式断点

```
xs: < 576px   (手机)     → 1列布局
sm: ≥ 576px   (平板竖屏) → 2列布局
md: ≥ 768px   (平板横屏) → 2列布局
lg: ≥ 992px   (小屏电脑) → 3列布局
xl: ≥ 1200px  (大屏电脑) → 4列布局
xxl: ≥ 1600px (超大屏)   → 4列布局
```

## 🌈 渐变背景

```css
background: linear-gradient(180deg, #f5f7fa 0%, #e8ecf1 100%);
```

这个渐变创造了从浅到深的微妙过渡，增加页面层次感。

## 💫 动画时长

```
卡片悬停: 0.3s ease
数据更新: 0.5s ease-in-out
页面加载: 0.3s ease
```

## 🎨 视觉层次

```
第一层: 顶部导航栏 (白色, 固定)
第二层: 快捷操作区 (白色卡片)
第三层: 核心指标卡片 (白色卡片 + 彩色边框)
第四层: 数据分析图表 (白色卡片)
背景层: 渐变背景
```

## 📏 最大宽度

```
内容区域最大宽度: 1600px
居中对齐: margin: 0 auto
```

这确保在超大屏幕上内容不会过度拉伸，保持最佳阅读体验。
