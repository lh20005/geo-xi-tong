# 标题生成技巧说明

## ✅ 已添加到提示词

## 📋 标题要求

### 基本要求
- **数量**：1个标题
- **字数**：18字以内
- **来源**：结合文章内容提炼
- **技巧**：使用以下4种技巧之一

## 🎯 四种标题技巧

### 1. 强调式标题

**特点**：
- 使用强调词制造紧迫感
- 利用损失厌恶心理
- 给读者"不看就亏了"的感觉

**强调词列表**：
- 最好、注意、必须、一定、千万
- 刚刚、不得了、震惊、震撼、炸锅

**参考范例**：
```
✅ 刚刚！1月浙A牌摇号结果新鲜出炉！速查有没有你
✅ 太敢了！撕开社会真相，这部飙升8.9的新片，到底什么来头！
✅ 突发！拼多多遭殃！"被迫"退出央视春晚独家
✅ 运营人，你必须知道的2021裂变技巧
```

**适用场景**：
- 时效性强的内容
- 重要通知或公告
- 必备知识或技能

### 2. 制造反差

**特点**：
- 在相邻元素间创建明显差异
- 打破常规认知
- 引发好奇和思考

**反差类型**：
- 行为与结果的反差
- 预期与现实的反差
- 常识与真相的反差

**参考范例**：
```
✅ 我被裁员了，但我更快乐了
✅ 我停止追求高薪，我的财富却意外增长了
✅ 我停止了节食，反而瘦了！
✅ 我戒掉了咖啡，精力更旺盛，睡眠更甜美
```

**适用场景**：
- 颠覆性观点
- 反常识内容
- 个人经历分享

### 3. 巧用数字

**特点**：
- 给人清晰、具体、易操作的感觉
- 快速吸引读者注意力
- 暗示内容的实用性

**数字用法**：
- 列举数量（5个方法、10个技巧）
- 具体数据（年薪35万、45岁）
- 时间节点（2021、1月）

**参考范例**：
```
✅ 初入职场，你必须要知道这10种办公室禁忌
✅ 挑选油烟机，你一定要知道这5个关键指标参数
✅ 老公年薪35万，我妈妈生病需住院，老公把银行卡给我，我看完余额却想离婚！
✅ 我45岁，离异，娶了35岁的漂亮剩女，婚后我就后悔了，她剩下是有原因的
```

**适用场景**：
- 教程类文章
- 清单类内容
- 数据分析类

### 4. 制造悬念

**特点**：
- 提出问题但不给答案
- 描述反常行为或异常事件
- 吸引读者好奇心

**悬念手法**：
- 提出疑问（为什么...？）
- 描述反常（放弃高薪、不想奋斗）
- 暗示真相（背后的真相、让我反思）

**参考范例**：
```
✅ 看了这本书以后，90后的我彻底不想奋斗了
✅ 他为什么放弃了月入十万的工作去旅行？背后的真相让我反思
✅ 看完这个视频，我决定放弃高薪工作
```

**适用场景**：
- 故事类文章
- 观点类内容
- 反思类文章

## 🎨 标题示例（按话题）

### 话题：英国留学机构哪家靠谱

**强调式**：
- 注意！选择英国留学机构，这3点必须看
- 千万别踩坑！英国留学机构选择指南

**制造反差**：
- 我换了3家留学机构，最后选了最便宜的
- 朋友花10万找中介，我DIY反而更成功

**巧用数字**：
- 对比5家英国留学机构，我发现这些秘密
- 英国留学机构选择，你必须知道的7个标准

**制造悬念**：
- 为什么大家都推荐这家留学机构？
- 我差点被留学机构坑了，幸好发现了这个

### 话题：Python如何读取Excel文件

**强调式**：
- 注意！Python读取Excel，这个方法最简单
- 必须掌握！Python处理Excel的3种方法

**制造反差**：
- 我用了5年Excel，才知道Python这么简单
- 放弃VBA吧，Python读取Excel更高效

**巧用数字**：
- Python读取Excel的3种方法，第2种最实用
- 5行代码搞定Excel读取，Python真香

**制造悬念**：
- 为什么大家都用Python处理Excel？
- 学会这招，我再也不手动处理Excel了
