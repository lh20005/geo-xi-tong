## 🔧 可能的问题

### 问题1：后端没有重启

**症状**：看不到详细的日志（Cookie数量、凭证数据等）

**解决**：
```bash
cd server
# 按 Ctrl+C 停止当前服务
npm run dev
```

### 问题2：登录检测失败

**症状**：浏览器关闭，但后端日志停在"等待用户登录"

**原因**：头条号登录后的URL不符合检测条件

**临时解决**：测试其他平台（知乎、简书）

### 问题3：Cookie为空

**症状**：日志显示"成功获取 0 个Cookie"

**原因**：登录检测太早，Cookie还没设置

**解决**：需要调整等待时间

### 问题4：数据库连接失败

**症状**：日志显示数据库错误

**检查**：
```bash
psql -d geo_system -c "SELECT 1;"
```

应该返回 `1`。
