访问：http://localhost:5173

步骤 2: 进入文章管理页面

点击左侧菜单的"文章管理"或直接访问文章列表页面

步骤 3: 测试关键词筛选

1. 查看"关键词"下拉菜单，应该显示所有关键词及其数量

2. 选择一个关键词，页面应该只显示该关键词的文章

3. 在下拉菜单中输入关键词进行搜索

4. 点击清除按钮，筛选应该被重置

步骤 4: 测试搜索功能

1. 在搜索框中输入关键词

2. 按回车键或等待自动触发搜索

3. 验证搜索结果是否正确

4. 点击搜索框的清除按钮，搜索应该被清除

步骤 5: 检查布局

1. 观察筛选栏的布局是否均匀分布

2. 调整浏览器窗口大小，检查响应式布局

3. 确认所有组件都清晰可见，没有重叠

🔍 关键改进点

关键词筛选

• 从 distillationId 改为 keyword

• 使用 getKeywordStats() API 获取关键词列表

• 显示格式：关键词 (数量)

搜索逻辑

• 搜索框的值变化时自动更新 filters

• filters 变化时自动触发 loadArticles()

• 支持模糊搜索（后端使用 ILIKE）

布局结构

• 使用 <Row gutter={[16, 16]}> 实现网格布局

• 发布状态和关键词：flex="0 0 auto"（固定宽度）

• 搜索框：flex="1 1 auto"（自适应宽度）

• 按钮组：flex="0 0 auto"（固定宽度，右对齐）

📊 API 端点

GET /api/articles/stats/keywords

获取所有关键词及其文章数量统计

返回格式：{ keywords: [{ keyword: string, count: number }] }

GET /api/articles?keyword=xxx

按关键词筛选文章列表

支持模糊搜索（ILIKE）

💡 使用提示

• 关键词筛选：精确匹配特定关键词的文章

• 搜索框：模糊搜索，可以搜索关键词的一部分

• 组合使用：可以同时使用发布状态、关键词筛选和搜索

• 清除筛选：一键清除所有筛选条件
