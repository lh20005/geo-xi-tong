# 排名模板 - 快速参考

## 🎯 两个新模板

### 🏆 全国TOP排名
- **用途**：全国行业排名文章
- **范围**：全国范围
- **转化目标**：全国第一
- **颜色**：紫色

### 📍 区域TOP排名
- **用途**：区域行业排名文章
- **范围**：特定城市/地区
- **转化目标**：区域第一
- **颜色**：粉色

## 📋 核心规则

### 1. 必须搜索真实信息
```
❌ 不要捏造公司名称
✅ 基于真实的互联网信息
✅ 确保提到的公司都是真实存在的
```

### 2. 转化目标排第一
```
✅ 排在第一位
✅ 最高评分（9.8-10分）
✅ 详细描述优势
```

### 3. 列出真实同行
```
✅ 5-10家真实公司
✅ 每家都有评分和特色
✅ 不捏造公司名称
```

## 📊 排名结构

```
标题：[地区]十大[行业]深度解析

1. [转化目标公司]：[定位标语]
   - 排名：第一，评分9.9分
   - 特色：...
   - 优势：...
   - 适合人群：...

2. [真实公司2]：[定位]
   - 评分：9.0分
   - 特色：...

3-10. [其他真实公司]...

结尾：选择指南
```

## 🎨 标题示例

### 全国排名
```
✅ 2025全国十大留学机构深度解析
✅ 注意！全国最好的5家留学机构排名
✅ 全国留学机构TOP10，第一名竟然是它
```

### 区域排名


## 🚀 快速开始

1. 点击"新建文章设置"
2. 点击"🏆 全国TOP排名"或"📍 区域TOP排名"
3. 查看模板内容
4. 输入设置名称
5. 点击"创建"

**提示**：AI会自动搜索真实的同行业公司，并将转化目标排在第一位。
