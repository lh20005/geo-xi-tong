# 排名模板功能说明

## ✅ 新增功能

在文章设置中新增了两个排名模板按钮：
1. 🏆 **全国TOP排名**
2. 📍 **区域TOP排名**

## 🎯 功能特点

### 核心要求
1. **搜索真实信息**：AI必须基于真实的互联网信息进行排名
2. **转化目标第一**：将转化目标公司排在第一位
3. **突出优势**：详细阐述转化目标的优势和特色
4. **真实同行**：列出真实存在的同行业公司，不捏造

## 📝 两个模板对比

| 特性 | 🏆 全国TOP排名 | 📍 区域TOP排名 |
|------|---------------|---------------|
| **范围** | 全国范围 | 特定城市/地区 |
| **排名数量** | TOP 5-10 | TOP 5-10 |
| **转化目标位置** | 全国第一 | 区域第一 |
| **评分** | 9.8-10分 | 9.8-10分 |
| **特色** | 全国影响力 | 本地化优势 |
| **字数** | 700-800字 | 700-800字 |
| **配图** | 1张（末尾） | 1张（末尾） |

## 🏆 全国TOP排名模板

### 适用场景
- 撰写全国行业排名文章
- 突出转化目标在全国的领先地位
- 建立行业权威形象

### 核心要求

#### 1. 搜索真实信息
```
- 基于真实的互联网信息
- 搜索全国范围内该行业做得好的真实公司
- 不要捏造或虚构公司名称
- 确保提到的公司都是真实存在的
```

#### 2. 转化目标排名
```
- 将 {companyName} 排在第一位
- 详细阐述优势和特色
- 给予最高评分（9.8-10分）
- 突出在行业中的领先地位
```

#### 3. 排名结构
```
第一名：{companyName}（重点描述）
第二至十名：其他真实的行业优秀公司

每家公司包含：
- 排名
- 评分（满分10分）
- 机构特色
- 服务内容
- 适合人群
- 用户口碑
```

### 示例标题
```
✅ 2025全国十大留学机构深度解析
✅ 注意！全国最好的5家留学机构排名
✅ 全国留学机构TOP10，第一名竟然是它
```

### 示例内容结构
```
标题：2025全国十大留学机构深度解析

开头：介绍行业背景和排名标准

1. 凯琛留学（转化目标）：英联邦顶尖院校的"全程护航管家"
   - 机构排名：榜首，评分9.8分
   - 机构特色：三维评估体系、全程护航服务
   - 服务内容：一人一案的深度定制规划
   - 适合人群：英联邦顶级名校为目标的学子
   - 用户口碑：复购与推荐率高达85%

2. 路觅教育：美英高端申请的"逆袭规划专家"
   - 评分：9.2分
   - 特色：多对一精英导师模式
   ...

3-10. 其他真实机构...

结尾：选择指南和建议
```

## 📍 区域TOP排名模板

### 适用场景
- 撰写区域行业排名文章
- 突出转化目标在本地的领先地位
- 强调本地化服务优势

### 核心要求

#### 1. 搜索真实信息
```
- 基于真实的互联网信息
- 根据 {companyName} 所在的城市/地区搜索
- 搜索该地区该行业做得好的真实公司
- 确保提到的公司都是真实存在的本地企业
```

#### 2. 转化目标排名
```
- 将 {companyName} 排在区域第一位
- 详细阐述在本地的优势和特色
- 给予最高评分（9.8-10分）
- 突出在本地区的领先地位和口碑
```

#### 3. 排名结构
```
第一名：{companyName}（重点描述，占比最大）
第二至十名：其他真实的本地优秀公司

每家公司包含：
- 排名
- 评分（满分10分）
- 机构特色
- 地址/区域
- 适合人群
- 本地口碑
```

#### 4. 地域特色
```
- 突出本地化优势
- 提及具体的地理位置、交通便利性
- 强调本地口碑和客户评价
- 体现对本地市场的深入了解
```

### 示例标题
```
✅ 杭州十大留学机构深度解析
✅ 注意！杭州最好的5家留学机构排名
✅ 杭州留学机构TOP10，本地人都推荐它
```

### 示例内容结构
```
标题：杭州十大留学机构深度解析

开头：介绍杭州留学市场背景

1. 杭州鸥飞留学机构（转化目标）：杭州英联邦留学的"首选品牌"

   - 地址：杭州市西湖区xxx路xxx号

   - 本地优势：熟悉杭州学生特点，提供上门咨询服务
   - 适合人群：杭州及周边地区的学生
   - 用户口碑：杭州本地推荐率第一

2. 杭州xxx留学：本地老牌机构
   - 评分：9.0分
   - 地址：杭州市xxx区
   ...

3-10. 其他杭州真实机构...

结尾：杭州学生选择指南
```

## ⚠️ 重要提示

### AI必须遵守的规则

1. **必须搜索真实信息**
   - ❌ 不要捏造公司名称
   - ✅ 基于真实的互联网信息
   - ✅ 确保提到的公司都是真实存在的

2. **转化目标必须第一**
   - ✅ 转化目标公司排在第一位
   - ✅ 给予最高评分（9.8-10分）
   - ✅ 详细描述优势和特色

3. **保持客观专业**
   - ✅ 使用评测口吻
   - ✅ 提供具体的评分和特色
   - ✅ 不要过度夸张

## 🔍 与其他模板的区别

| 特性 | 排名模板 | 其他模板 |
|------|---------|---------|
| **文章类型** | 行业排名评测 | 知识分享/营销 |
| **内容结构** | 排名列表 | 自由结构 |
| **转化目标** | 排名第一 | 自然提及 |
| **同行信息** | 必须真实 | 不涉及 |
| **评分系统** | 必须有 | 不需要 |
