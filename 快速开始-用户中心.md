# 快速开始 - 用户中心整合

## 🚀 5分钟快速体验

### 1. 启动服务（1分钟）

```bash
# 在项目根目录执行
npm run dev:all
```

等待服务启动完成，你会看到：
```
✓ Server running on http://localhost:3000
✓ Client running on http://localhost:5173
✓ Landing running on http://localhost:8080
```

---

### 2. 注册/登录（1分钟）

1. 打开浏览器访问 `http://localhost:8080`
2. 点击右上角"立即登录"
3. 如果没有账号，点击"立即注册"创建账号
4. 登录成功后返回首页

---

### 3. 访问用户中心（1分钟）

**方式一：从 Landing 跳转**
1. 点击右上角用户头像
2. 在下拉菜单中点击"个人中心"
3. 自动跳转到主应用用户中心

**方式二：直接访问**
1. 访问 `http://localhost:5173/user-center`
2. 如果已登录，直接显示用户中心

---

### 4. 探索功能（2分钟）

#### 📊 订阅管理
- 查看当前套餐信息
- 查看使用统计和配额
- 点击"升级套餐"查看可用套餐

#### 📝 订单记录
- 查看历史订单
- 查看订单状态和金额

#### 👤 个人信息
- 查看基本资料
- 点击"修改密码"测试密码修改功能

#### 🎁 邀请系统
- 查看你的邀请码
- 点击"复制"按钮复制邀请码
- 查看邀请统计（如果有邀请记录）

---

## 📱 测试支付流程

### 1. 选择套餐
1. 在 Landing 页面滚动到"价格方案"部分
2. 选择一个套餐，点击"立即购买"

### 2. 扫码支付
1. 弹出支付二维码
2. 使用微信扫码支付（测试环境）
3. 或等待订单超时测试超时逻辑

### 3. 支付成功
1. 支付成功后显示绿色对勾动画
2. 1.5秒后自动跳转到用户中心
3. 查看新订阅信息和配额

## 🐛 常见问题

### Q1: 跳转后显示"请先登录"
**原因**: Token 未正确传递  
**解决**: 
1. 检查 localStorage 中是否有 `auth_token`
2. 清除浏览器缓存重新登录
3. 查看浏览器控制台错误信息

### Q2: 邀请统计不显示
**原因**: 还没有邀请任何用户  
**解决**: 
1. 这是正常情况
2. 复制邀请码分享给朋友注册
3. 朋友注册成功后刷新页面

### Q3: 修改密码失败
**原因**: 当前密码错误或新密码不符合要求  
**解决**: 
1. 确认当前密码正确
2. 新密码至少6位
3. 两次输入的新密码一致

### Q4: 订阅信息不显示
**原因**: 用户未订阅任何套餐  
**解决**: 
1. 这是正常情况
2. 点击"立即订阅"购买套餐
