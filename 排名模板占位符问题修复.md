# 排名模板占位符问题修复

## ✅ 问题修复

修复了全国TOP和区域TOP排名模板中AI输出占位符文字的问题。

## 🎯 问题描述

**问题**：
生成的文章中出现了类似以下的占位符文字：
```
【开头段落】
（空一行）
[开头段落：介绍行业背景和排名标准]
[结尾段落：选择指南和建议]
```

**原因**：
AI把提示词中的格式说明当成了要输出的内容。

## ✅ 解决方案

### 1. 修改输出格式部分

**修改前**：
```
【输出格式】
标题：[文章标题]

[开头段落：介绍行业背景和排名标准]

（空一行）

1. {companyName}：[定位标语]
...
```

**修改后**：
```
【输出格式示例】
注意：以下是格式示例，不要输出方括号[]和括号()中的说明文字

标题：2025全国十大留学机构深度解析

面对日益激烈的海外名校申请竞争，选择一个专业、可靠的留学机构...

1. {companyName}：英联邦顶尖院校的"全程护航管家"

机构排名：全国榜首，评分9.8分
机构特色：独创的"三维评估体系"...
```

### 2. 修改排版格式要求

**修改前**：
```
【排版格式要求 - 非常重要】
每家公司的信息必须分段展示，格式如下：

1. [公司名称]：[定位标语]

机构排名：[排名和评分]
机构特色：[核心特色描述]
...

（空一行）
```

**修改后**：
```
【排版格式要求 - 非常重要】
每家公司的信息必须分段展示，使用以下标签：

- 公司名称和定位标语作为小标题
- 机构排名：写明排名和评分
- 机构特色：描述核心特色
...

每家公司之间必须空一行分隔
```

## 🎯 关键改进

### 1. 使用具体示例代替占位符

**原则**：
- ❌ 不要用 `[开头段落：...]` 这样的占位符
- ✅ 直接写出具体的示例内容

**示例**：
```
❌ 错误：
[开头段落：介绍行业背景和排名标准]

✅ 正确：
面对日益激烈的海外名校申请竞争，选择一个专业、可靠的留学机构已成为成功的关键一环...
```

### 2. 明确说明不要输出占位符

在输出格式部分添加明确说明：
```
注意：以下是格式示例，不要输出方括号[]和括号()中的说明文字
```

### 3. 简化排版要求

**原则**：
- ❌ 不要用带占位符的格式示例
- ✅ 用列表说明需要的标签

**示例**：
```
❌ 错误：
1. [公司名称]：[定位标语]
机构排名：[排名和评分]
（空一行）

✅ 正确：
使用以下标签：
- 公司名称和定位标语作为小标题
- 机构排名：写明排名和评分
每家公司之间必须空一行分隔
```

## 💡 经验总结

### 提示词编写原则

1. **使用具体示例**
   - 不要用抽象的占位符
   - 直接写出期望的输出内容

2. **明确说明**
   - 告诉AI什么不要输出
   - 使用"注意："、"不要..."等明确指令

3. **避免歧义**
   - 不要用方括号[]包裹说明文字
   - 不要用括号()包裹格式说明

4. **测试验证**
   - 生成文章后检查是否有占位符
   - 确保AI理解了格式要求
