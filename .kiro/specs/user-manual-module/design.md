# Design Document: 使用说明书页面

## Overview

使用说明书页面是一个展示系统使用教程的静态内容页面。页面将包含系统简介、快速开始指南、各功能模块的详细使用步骤、配套截图等内容，帮助新用户快速掌握系统的使用方法。

```
│         UserManualPage                   │
│  ┌───────────────────────────────────┐  │
│  │  目录导航 (Anchor)                 │  │
│  └───────────────────────────────────┘  │

│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

## Data Models

说明书内容将直接写在组件中，使用以下结构组织：

1. **系统简介** - 介绍系统的主要功能和价值
2. **快速开始** - 展示典型的端到端工作流程
3. **功能模块** - 按照导航菜单顺序，详细说明每个功能的使用方法

   - 文章管理
   - 平台登录
   - 发布任务
   - 发布记录
   - 系统配置
4. **常见问题** - 解答用户可能遇到的问题

## 说明书内容大纲

### 1. 系统简介
- 系统概述
- 主要功能
- 适用场景

### 2. 快速开始
- 第一步：配置AI API
- 第二步：设置转化目标（可选）
- 第三步：关键词蒸馏
- 第四步：生成文章
- 第五步：配置发布平台
- 第六步：创建发布任务

### 3. 功能模块详细说明

#### 3.1 工作台
- 功能概述：查看系统运营数据和快速入口
- 使用说明：
  - 查看核心指标卡片
  - 查看趋势图表
  - 点击卡片快速跳转到对应功能

#### 3.2 转化目标
- 功能概述：设置企业信息，用于文章生成时引用
- 使用步骤：
  1. 点击"新建转化目标"按钮
  2. 填写公司名称（必填）
  3. 填写行业类型、官方网站、公司地址（可选）
  4. 点击"创建"按钮保存
- 注意事项：转化目标信息会在生成文章时自动引用

#### 3.3 关键词蒸馏
- 功能概述：输入关键词，AI生成相关话题
- 使用步骤：
  1. 在输入框中输入关键词（如"英国留学"）
  2. 点击"开始蒸馏"按钮
  3. 等待AI生成话题（约10-30秒）
  4. 系统自动跳转到蒸馏结果页面
- 注意事项：
  - 使用前需先配置AI API
  - 关键词应该是您的目标业务领域

#### 3.4 蒸馏结果
- 功能概述：查看和管理蒸馏生成的话题
- 使用步骤：
  1. 查看生成的话题列表
  2. 勾选需要生成文章的话题
  3. 点击"生成文章"按钮
- 操作说明：
  - 可以编辑话题内容
  - 可以删除不需要的话题
  - 可以批量选择话题

#### 3.5 企业图库
- 功能概述：管理企业图片资源
- 使用步骤：
  1. 点击"创建相册"按钮
  2. 输入相册名称
  3. 上传图片到相册
  4. 图片可在生成文章时使用

#### 3.6 企业知识库
- 功能概述：管理企业知识内容
- 使用步骤：
  1. 点击"新建知识库"按钮
  2. 输入知识库标题和内容
  3. 保存后可在生成文章时引用

#### 3.7 文章设置
- 功能概述：配置文章生成的参数
- 使用步骤：
  1. 设置文章生成提示词
  2. 配置文章长度、风格等参数
  3. 点击"保存配置"按钮

#### 3.8 生成文章
- 功能概述：创建文章生成任务
- 使用步骤：
  1. 点击"新建任务"按钮
  2. 选择转化目标（可选）
  3. 选择关键词
  4. 选择蒸馏结果（话题）
  5. 设置生成数量
  6. 点击"创建任务"按钮
  7. 等待任务执行完成
- 注意事项：
  - 任务会自动执行，无需手动触发
  - 可以查看任务进度和生成的文章数量

#### 3.9 文章管理
- 功能概述：查看、编辑和管理生成的文章
- 使用步骤：
  1. 在列表中查看所有文章
  2. 点击"预览"查看文章内容
  3. 点击"编辑"修改文章
  4. 点击"删除"删除文章
- 筛选功能：
  - 按关键词筛选
  - 按发布状态筛选
  - 搜索文章标题或内容

#### 3.10 平台登录
- 功能概述：配置发布平台账号
- 使用步骤：
  1. 点击平台卡片（如"头条号"）
  2. 系统自动打开浏览器登录页面
  3. 在浏览器中完成登录
  4. 登录成功后Cookie自动保存
  5. 返回系统，平台卡片显示"已登录"
- 注意事项：
  - 需要有对应平台的账号
  - 登录信息会自动保存，无需重复登录
  - 支持多个平台同时登录

#### 3.11 发布任务
- 功能概述：创建文章发布任务
- 使用步骤：
  1. 在"选择文章"区域勾选要发布的文章
  2. 在"选择发布平台"区域勾选目标平台
  3. 设置发布间隔（分钟）
  4. 选择是否开启静默发布模式
  5. 点击"创建发布任务"按钮
  6. 确认任务信息后点击"确定"
- 任务管理：
  - 查看任务状态（等待中、执行中、成功、失败）
  - 点击"日志"查看执行详情
  - 点击"执行"立即执行等待中的任务
  - 点击"终止"停止正在执行的任务
  - 点击"删除"删除已完成的任务
- 注意事项：
  - 使用前需先在"平台登录"中登录对应平台
  - 发布间隔建议设置5分钟以上，避免被平台限制
  - 静默模式下浏览器在后台运行，不会弹出窗口

#### 3.12 发布记录
- 功能概述：查看历史发布记录
- 使用说明：
  - 查看所有发布任务的历史记录
  - 筛选成功/失败的记录
  - 查看发布时间和平台信息

### 4. 常见问题

#### Q1: 如何获取AI API Key？
A: 
- DeepSeek: 访问 https://platform.deepseek.com 注册并获取
- Gemini: 访问 https://makersuite.google.com/app/apikey 获取
- Ollama: 本地安装，无需API Key

#### Q2: 关键词蒸馏失败怎么办？
A: 
1. 检查是否已配置AI API
2. 检查API Key是否正确
3. 检查网络连接是否正常
4. 查看错误提示信息

#### Q3: 文章生成失败怎么办？
A: 
1. 检查AI API配置是否正确
2. 检查是否选择了蒸馏结果
3. 查看任务日志了解具体错误

#### Q4: 平台登录失败怎么办？
A: 
1. 确保浏览器已安装
2. 检查网络连接
3. 尝试手动在浏览器中登录平台
4. 查看系统日志了解详细错误

#### Q5: 发布任务一直等待中？
A: 
1. 检查是否有其他任务正在执行
2. 系统采用串行发布，需等待前面的任务完成
3. 可以点击"执行"按钮立即执行

#### Q6: 如何批量发布文章？
A: 
1. 在发布任务页面勾选多篇文章
2. 勾选多个发布平台
3. 系统会自动创建多个任务
4. 任务会按照设置的间隔依次执行

## Implementation Notes

### 1. 页面结构
- 使用Ant Design的Card、Typography、Anchor等组件
- 使用响应式布局，支持桌面和移动端
- 使用Anchor组件实现目录导航

### 2. 内容组织
- 内容直接写在组件中，使用JSX/TSX
- 使用Typography组件（Title、Paragraph、Text）排版
- 使用List、Steps等组件展示步骤
- 使用Alert组件展示注意事项

### 3. 截图管理
- 截图存储在 `client/public/manual-screenshots/` 目录
- 使用Image组件展示截图
- 截图添加边框和阴影效果
- 支持点击放大查看（使用Image.PreviewGroup）

### 4. 样式设计
- 使用Ant Design默认主题
- 添加适当的间距和分隔线
- 使用不同的标题层级区分章节
- 代码块使用等宽字体

### 5. 搜索功能
- 在页面顶部添加搜索输入框
- 实现页面内文本搜索和高亮
- 支持跳转到下一个/上一个匹配
- 显示匹配数量
- 使用mark.js或自定义实现高亮功能

### 6. 打印支持
- 添加打印按钮
- 使用window.print()触发打印
- 添加@media print样式优化打印效果
