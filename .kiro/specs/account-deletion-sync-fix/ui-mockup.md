## 设计规范

### 颜色方案

- **主色调**: #1890ff (蓝色)
- **渐变背景**: #667eea → #764ba2 (紫色渐变)
- **卡片背景**: #ffffff (白色)
- **文字颜色**: 
  - 主要文字: #1e293b
  - 次要文字: #64748b
  - 错误提示: #ff4d4f

### 组件尺寸

- **登录卡片**: 400px 宽度
- **输入框**: 大尺寸 (size="large")
- **Logo图标**: 80x80px
- **圆角**: 8px
- **阴影**: 0 10px 40px rgba(0,0,0,0.1)

### 交互效果

1. **输入框聚焦**: 蓝色边框高亮
2. **按钮悬停**: 颜色加深
3. **加载状态**: 按钮显示loading动画
4. **错误提示**: 红色文字 + 抖动动画

### 响应式设计

- **桌面端**: 居中显示，400px卡片
- **平板**: 适应屏幕宽度，最大400px
- **手机**: 全屏显示，边距16px

## 用户流程

1. **访问任何页面（未登录）** → 自动跳转到登录页
2. **输入用户名和密码** → 点击登录按钮
3. **验证成功** → 显示欢迎消息 → 跳转到首页
4. **验证失败** → 显示错误提示 → 保持在登录页

## 安全提示

- 密码输入框使用 `Input.Password` 组件（可切换显示/隐藏）
- 表单验证：用户名最少3个字符，密码最少6个字符
- 登录失败不显示具体原因（用户名或密码错误）
- 默认账号仅用于开发测试，生产环境应移除提示
