# 文章设置模板功能说明

## 功能概述

在文章设置页面，点击"新建文章设置"按钮后，弹出的对话框中新增了**三个快速模板按钮**，用户可以一键应用预设的提示词模板。

## 三种模板

### 1. ⚖️ 平衡型

**适用场景**：既要专业内容，又要适度营销

**特点**：
- 专业内容与适度营销相结合
- 公司名称出现1-2次
- 保持专业性，避免过度营销

**模板内容**：
```
你是一个专业的{keyword}领域专家。

【用户关心的问题】
{topics}

【写作要求】
1. 围绕关键词"{keyword}"展开，确保内容专业、有价值
2. 详细回答用户的每个问题，提供实用建议
3. 在合适的位置适度提及{companyName}（1-2次即可）
4. 参考知识库内容，确保信息准确：
{knowledgeBase}
5. 字数：1500-2500字
6. 语言：专业、流畅、易懂

【输出格式】
严格按照以下格式输出：

标题：[文章标题]

[文章正文内容]

【注意事项】
- 不要输出思考过程
- 不要使用Markdown符号（#、*、-等）
- 保持专业性，避免过度营销
- 公司名称只在1-2个自然位置出现
```

### 2. 🎓 重专业

**适用场景**：纯专业内容，不做营销推广

**特点**：
- 深度专业分析
- 不提及公司名称
- 纯粹的专业性和客观性
- 字数更多（2000-3000字）

**模板内容**：
```
你是一个资深的{keyword}领域专家和内容创作者。

【核心任务】
撰写一篇关于"{keyword}"的深度专业文章，回答以下用户关心的问题：
{topics}

【内容要求】
1. 专业深度：
   - 提供深入的专业分析和见解
   - 使用准确的专业术语
   - 给出具体的数据和案例支持

2. 结构清晰：
   - 开头：引入话题，说明重要性
   - 中间：详细回答每个问题，层层递进
   - 结尾：总结要点，给出实用建议

3. 内容质量：
   - 围绕关键词"{keyword}"深入展开
   - 自然融入话题：{topicsList}
   - 参考知识库内容，确保准确性：
{knowledgeBase}
   - 字数：2000-3000字

4. 语言风格：
   - 专业、严谨、客观
   - 逻辑清晰、论证充分
   - 通俗易懂，避免过于晦涩

【输出格式】
标题：[文章标题]

[文章正文内容]

【禁止事项】
- 不要输出思考过程或分析步骤
- 不要使用Markdown格式符号
- 不要添加营销性质的内容
- 保持纯粹的专业性和客观性
```

### 3. 📢 重营销

**适用场景**：以营销推广为主要目的

**特点**：
- 营销推广为主
- 公司名称出现3-4次
- 引导用户行动
- 突出品牌优势

**模板内容**：
```
你是一个专业的营销文案撰写专家，擅长将专业内容与品牌推广完美结合。

【核心关键词】{keyword}

【用户关心的问题】
{topics}

【品牌信息】
重点推广：{companyName}
企业优势和特色：
{knowledgeBase}

【写作策略】
1. 内容价值：
   - 围绕"{keyword}"提供有价值的专业内容
   - 详细回答用户问题，建立信任
   - 展示专业性和权威性

2. 品牌植入：
   - 在文章中多次自然提及{companyName}（3-4次）
   - 在介绍解决方案时重点推荐{companyName}
   - 在给出建议时引导用户关注{companyName}
   - 结尾处强化{companyName}的优势和价值

3. 营销技巧：
   - 使用对比手法，突出{companyName}的优势
   - 引用成功案例或用户评价
   - 制造紧迫感，引导用户行动
   - 提供明确的行动指引

4. 内容要求：
   - 话题融入：{topicsList}
   - 字数：1500-2000字
   - 语言：专业但不失亲和力

【输出格式】
标题：[吸引人的文章标题]

[文章正文内容]

【注意事项】
- 营销要自然，不要过于生硬
- 保持内容的专业性和可信度
- 不要使用Markdown符号
- 不要输出思考过程
```

## 使用方式

### 步骤1：打开新建对话框

点击文章设置页面的"新建文章设置"按钮

### 步骤2：选择模板

在提示词输入框下方，有三个模板按钮：

- **⚖️ 平衡型** - 蓝色按钮
- **🎓 重专业** - 绿色按钮
- **📢 重营销** - 橙色按钮

### 步骤3：一键应用

点击任意模板按钮，对应的提示词会自动填充到输入框中

### 步骤4：自定义（可选）

- 如果满意，直接保存
- 如果需要调整，可以在模板基础上修改
- 也可以完全自己编写，不使用模板

### 步骤5：保存

输入设置名称，点击"创建"按钮保存

|------|--------|--------|--------|
| 公司名称出现次数 | 1-2次 | 0次 | 3-4次 |
| 营销程度 | 适度 | 无 | 强 |
| 专业深度 | 中等 | 深 | 中等 |
| 字数 | 1500-2500 | 2000-3000 | 1500-2000 |
| 适用场景 | 日常内容 | 专业文章 | 推广文案 |

## 技术实现

### 1. 模板常量文件

**位置**：`client/src/constants/promptTemplates.ts`

**特点**：
- 模板内容不可修改（常量）
- 永远可以一键调用
- 集中管理，易于维护

### 2. 组件修改

**位置**：`client/src/components/ArticleSettingModal.tsx`

**新增功能**：
- 三个模板按钮
- 点击按钮自动填充提示词
- 显示模板描述（Tooltip）
- 显示占位符提示

### 3. 用户体验

- ✅ 一键应用，无需手动复制粘贴
- ✅ 可以在模板基础上修改
- ✅ 也可以完全自定义
- ✅ 模板永远可用，不会丢失

## 占位符说明

所有模板都使用了占位符变量：

- `{keyword}` - 核心关键词
- `{topics}` - 话题列表（带编号）
- `{topicsList}` - 话题列表（逗号分隔）
- `{companyName}` - 公司名称
- `{knowledgeBase}` - 知识库内容

这些占位符在生成文章时会被自动替换为实际内容。

## 最佳实践

### 场景1：日常内容生成

**推荐**：平衡型

**原因**：
- 既有专业内容，又有适度营销
- 不会过度推销，保持可信度
- 适合大多数场景

### 场景2：建立专业形象

**推荐**：重专业

**原因**：
- 纯专业内容，建立权威性
- 不做营销，更容易获得信任
- 适合行业分析、技术文章

### 场景3：产品推广

**推荐**：重营销

**原因**：
- 明确的营销目的
- 多次提及公司，强化品牌
- 引导用户行动

### 场景4：自定义需求

**推荐**：选择最接近的模板，然后修改

**原因**：
- 模板提供了良好的结构
- 在模板基础上修改更高效
- 保证了基本的质量

## 注意事项

1. **模板不可修改**：
   - 模板内容是固定的，不能在代码中修改
   - 如需调整，在应用模板后手动修改

2. **可以混合使用**：
   - 可以从一个模板开始
   - 然后参考另一个模板的部分内容
   - 创建自己的混合版本

3. **保存后可编辑**：
   - 保存的文章设置可以随时编辑
   - 编辑时仍然可以重新应用模板

4. **模板只是起点**：
   - 模板提供了基础结构
   - 鼓励根据实际需求调整
   - 创造最适合自己的版本
