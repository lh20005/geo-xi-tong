# 知识库功能 - 快速使用指南

## 功能概述
知识库功能允许您上传和管理文档，用于文章生成时的参考资料。

## 快速开始

### 1. 创建知识库
```
1. 打开 Windows 登录管理器
2. 点击左侧菜单"知识库"
3. 点击"创建知识库"按钮
4. 输入知识库名称（例如："装修"）
5. 输入描述（可选）
6. 点击"确定"
```

### 2. 上传文档
```
1. 点击知识库卡片进入详情页
2. 点击"上传文档"按钮
3. 选择要上传的文件（支持多选）
4. 点击"上传"
```

**支持的文件格式**:
- `.txt` - 纯文本文件
- `.md` - Markdown 文件
- `.pdf` - PDF 文档
- `.doc` - Word 文档（旧版）
- `.docx` - Word 文档（新版）

**文件限制**:
- 单个文件最大 10MB
- 一次最多上传 20 个文件

### 3. 查看文档
```
1. 在知识库详情页，文档列表会显示所有已上传的文档
2. 点击"查看"按钮可以查看文档内容
3. 在查看窗口中可以复制文档内容
```

### 4. 搜索文档
```
1. 在知识库详情页顶部的搜索框中输入关键词
2. 点击搜索按钮或按回车
3. 系统会显示包含关键词的文档
```

### 5. 删除文档
```
1. 在文档列表中找到要删除的文档
2. 点击"删除"按钮
3. 确认删除操作
```

### 6. 编辑知识库
```
1. 在知识库列表页，点击知识库卡片上的"编辑"按钮
2. 修改名称或描述
3. 点击"确定"保存
```

### 7. 删除知识库
```
1. 在知识库列表页，点击知识库卡片上的"删除"按钮
2. 确认删除操作
3. 注意：删除知识库会同时删除其中的所有文档
```

## 最佳实践

### 1. 文档命名
- ✅ 使用清晰的文件名: `装修流程-2024.txt`
- ❌ 避免模糊的文件名: `文档1.txt`

### 2. 文档组织
- 按主题创建不同的知识库
- 每个知识库专注于一个领域
- 定期更新过时的文档

### 3. 文档内容
- 使用 UTF-8 编码保存文本文件
- 保持文档内容简洁明了
- 避免上传重复内容

### 4. 文档大小
- 大文件建议拆分成多个小文件
- 图片较多的 PDF 可能超过大小限制
- 考虑使用压缩工具减小文件大小

## 常见问题

### Q1: 为什么上传失败？
**可能原因**:
1. 文件格式不支持 → 检查文件扩展名
2. 文件太大 → 压缩或拆分文件
3. 网络问题 → 检查网络连接
4. 知识库不存在 → 刷新页面重试

### Q2: 为什么看不到其他用户的知识库？
**答**: 这是正常的。每个用户只能看到和管理自己的知识库，这是为了保护数据隐私。

### Q3: 可以创建同名知识库吗？
**答**: 不可以。同一用户不能创建同名知识库，但不同用户可以创建同名知识库。

### Q4: 上传的文档存储在哪里？
**答**: 文档内容存储在数据库中，临时文件会在处理后自动删除。

### Q5: 如何批量上传文档？
**答**: 在文件选择对话框中，按住 Ctrl（Windows）或 Cmd（Mac）键，可以选择多个文件。

### Q6: 支持哪些文档格式？
**答**: 目前支持 .txt, .md, .pdf, .doc, .docx 格式。

### Q7: 文档内容会被其他用户看到吗？
**答**: 不会。系统有严格的多租户隔离，每个用户只能访问自己的知识库和文档。

### Q8: 可以导出知识库吗？
**答**: 目前可以查看和复制单个文档内容。批量导出功能计划在未来版本中添加。
