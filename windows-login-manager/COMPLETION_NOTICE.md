### 🚀 快速开始

#### 启动后端服务器
```bash
cd server
npm install
npm run dev
```

#### 启动Electron应用
```bash
cd windows-login-manager
npm install
npm run electron:dev
```

#### 构建安装包
```bash
cd windows-login-manager
npm run build:win
```
