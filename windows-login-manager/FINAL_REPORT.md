## 📝 使用说明

### 立即开始

```bash
# 1. 安装依赖
cd windows-login-manager
npm install

# 2. 启动开发环境
npm run electron:dev

# 3. 构建生产版本（需要先添加图标）
npm run build:win
```

### 重要提示

⚠️ **构建前需要**: 添加`build/icon.ico`文件  
📖 **详细说明**: 查看`BUILD_INSTRUCTIONS.md`
