## 格式说明

### PNG vs JPEG

| 格式 | 优势 | 劣势 | 使用场景 |
|------|------|------|---------|
| PNG | 支持透明背景、无损压缩 | 文件较大 | Logo、需要透明背景的图标 |
| JPEG | 文件更小、加载更快 | 不支持透明背景 | 照片、不需要透明的图标 |

### 优化建议
头条号图标较大（252KB），建议压缩到50KB以下：
```bash
# 使用 imagemin 压缩
imagemin toutiaohao.png --plugin=pngquant > toutiaohao-optimized.png

# 或转换为 JPEG
convert toutiaohao.png -quality 80 toutiaohao.jpeg
```

## 三级回退机制

每个平台都有完善的回退策略：

```
1. 本地图标 (/images/xxx.png 或 .jpeg)
   ↓ 加载失败
2. 在线图标 (CDN URL)
   ↓ 加载失败
3. 首字母显示 (如"百"、"头"、"小"、"微"、"抖"、"搜")
```
