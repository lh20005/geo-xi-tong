# 平台图标说明

这个目录存放各个平台的图标文件。

## 图标规格
- 尺寸: 48x48px 或更高（推荐 96x96px 以支持高清屏）
- 格式: PNG 或 SVG
- 命名: 使用平台ID作为文件名，如 `toutiao.png`

## 支持的平台
- toutiao.png - 头条号
- weixin.png - 微信公众号
- zhihu.png - 知乎
- douyin.png - 抖音
- xiaohongshu.png - 小红书
- bilibili.png - B站
- kuaishou.png - 快手
- baidu.png - 百度
- sohu.png - 搜狐
- sina.png - 新浪
- wangyi.png - 网易
- qq.png - 腾讯
- taobao.png - 淘宝
- jd.png - 京东
- pinduoduo.png - 拼多多

## 如何添加新平台图标
1. 准备48x48px或更高分辨率的PNG/SVG图标
2. 将文件命名为平台ID（如 `newplatform.png`）
3. 放入此目录
4. 图标会自动被识别和使用
