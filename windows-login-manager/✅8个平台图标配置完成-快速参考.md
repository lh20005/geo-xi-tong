## 使用方法

### 本地开发
```bash
cd windows-login-manager
npm run dev
# 访问 http://localhost:5174/platform-management
```

### 生产构建
```bash
npm run build
# 所有图标自动复制到 dist/images/
```

### 部署
直接部署 `dist` 目录，图标会自动可用：
```
https://your-domain.com/images/baijiahao.png
https://your-domain.com/images/toutiaohao.png
https://your-domain.com/images/xiaohongshu.png
https://your-domain.com/images/gongzhonghao.png
https://your-domain.com/images/douyin.jpeg
https://your-domain.com/images/souhu.jpeg
https://your-domain.com/images/wangyi.png
https://your-domain.com/images/bili.png
```

## 格式说明

- **6个PNG**: 百家号、头条号、小红书、微信公众号、网易号、哔哩哔哩
- **2个JPEG**: 抖音、搜狐号

PNG和JPEG混合使用完全没问题，浏览器自动识别。

## 性能建议

头条号图标较大（252KB），建议优化：
```bash
# 压缩PNG
imagemin toutiaohao.png --plugin=pngquant > toutiaohao-optimized.png

# 或转换为JPEG
convert toutiaohao.png -quality 80 toutiaohao.jpeg
```
