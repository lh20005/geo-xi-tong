# Windows 安装指南

## 📦 安装包信息

- **文件名**: `平台登录管理器 Setup 1.0.0.exe`
- **文件大小**: 74 MB
- **位置**: `windows-login-manager/release/`
- **支持系统**: Windows 7/8/10/11 (64位)

## 🚀 安装步骤

### 1. 下载安装包
从 `release/` 文件夹获取 `平台登录管理器 Setup 1.0.0.exe`

### 2. 运行安装程序
双击 `.exe` 文件开始安装

### 3. 处理 Windows SmartScreen 警告

**⚠️ 重要：由于应用未签名，会出现安全警告**

#### 如果看到 "Windows 已保护你的电脑" 提示：

1. 点击 **"更多信息"**
2. 点击 **"仍要运行"**
3. 继续安装

![SmartScreen 警告示例]
```
Windows 已保护你的电脑
Windows Defender SmartScreen 已阻止启动一个未识别的应用。
运行此应用可能会导致你的电脑存在风险。

[更多信息]  [不运行]

点击"更多信息"后：
[仍要运行]
```

**为什么会出现这个警告？**
- 应用未购买代码签名证书（约 $100-300/年）
- 这是正常现象，不代表应用有问题
- 适用于内部测试和小范围使用

### 4. 选择安装选项

- **安装位置**: 可以自定义安装路径
- **桌面快捷方式**: 建议勾选
- **开始菜单快捷方式**: 建议勾选

### 5. 完成安装

安装完成后，应用会自动启动

## 🔧 使用前准备

### 确保后端服务运行

应用需要连接到后端 API 服务：

```bash
# 在主项目目录运行
npm run server:dev
```

后端服务地址：`http://localhost:3000`

### 配置说明

首次启动时，应用会使用默认配置：
- API 地址: `http://localhost:3000`
- 可在设置页面修改

## ❓ 常见问题

### Q1: 安装时提示"未知发布者"
**A**: 这是正常的，因为应用未签名。按照上述步骤点击"仍要运行"即可。

### Q2: 杀毒软件报警
**A**: 部分杀毒软件可能误报。可以添加到白名单，或暂时关闭杀毒软件安装。

### Q3: 无法连接后端
**A**: 确保后端服务在 `http://localhost:3000` 运行。检查防火墙设置。

### Q4: 应用无法启动
**A**: 
1. 检查是否安装了 .NET Framework（Windows 7/8 需要）
2. 尝试以管理员身份运行
3. 查看日志文件（在用户目录下）

### Q5: 如何卸载
**A**: 
- 方式1: 控制面板 → 程序和功能 → 卸载
- 方式2: 开始菜单 → 平台登录管理器 → 卸载
