## 删除按钮详细说明

### 按钮组布局
```
┌────────────────────────────────────────────────────────────────┐
│                                                                 │
│  [🗑️ 删除选中(3)]   [🗑️ 删除当前关键词]   [🗑️ 删除筛选结果] │
│   ↑ 普通红色按钮      ↑ 普通红色按钮         ↑ 主要红色按钮   │
│   显示选中数量        需要关键词筛选         需要任意筛选条件  │
│                                                                 │
└────────────────────────────────────────────────────────────────┘
```

### 按钮状态

#### 1. 删除选中按钮
```
有选中项时：
┌──────────────────┐
│ 🗑️ 删除选中(3)   │  ← 红色，可点击
└──────────────────┘

无选中项时：
┌──────────────────┐
│ 🗑️ 删除选中(0)   │  ← 灰色，禁用
└──────────────────┘
```

#### 2. 删除当前关键词按钮
```
有关键词筛选时：
┌────────────────────────┐
│ 🗑️ 删除当前关键词      │  ← 红色，可点击
└────────────────────────┘

无关键词筛选时：
┌────────────────────────┐
│ 🗑️ 删除当前关键词      │  ← 灰色，禁用
└────────────────────────┘
```

#### 3. 删除筛选结果按钮
```
有筛选条件时：
┌────────────────────────┐
│ 🗑️ 删除筛选结果        │  ← 深红色主按钮，可点击
└────────────────────────┘

无筛选条件时：
┌────────────────────────┐
│ 🗑️ 删除筛选结果        │  ← 灰色，禁用
└────────────────────────┘
```

## 键盘快捷键（建议）

- `Delete` 或 `Backspace`：删除选中项
- `Ctrl + Shift + K`：删除当前关键词
- `Ctrl + Shift + D`：删除筛选结果

## 无障碍支持

- 所有按钮都有明确的 `aria-label`
- 禁用状态有 `aria-disabled` 属性
- 支持键盘导航（Tab键）
- 支持屏幕阅读器

```html
<Button
  danger
  icon={<DeleteOutlined />}
  onClick={handleDeleteSelected}
  disabled={selectedRowKeys.length === 0}
  aria-label={`删除选中的 ${selectedRowKeys.length} 项`}
  aria-disabled={selectedRowKeys.length === 0}
>
  删除选中 ({selectedRowKeys.length})
</Button>
```
