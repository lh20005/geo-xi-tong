# 发布任务模块 - README

## 🎯 模块简介

发布任务模块是一个全新的功能页面，位于左侧边栏"平台登录"下方。该模块整合了文章选择、平台选择和任务管理三大核心功能，为用户提供了一个完整的文章发布工作流。

## ✨ 核心功能

### 1️⃣ 文章选择
- 自动加载草稿状态的文章
- 支持单选和全选
- 表格形式清晰展示

### 2️⃣ 平台选择
- 以小卡片形式展示已登录平台
- 点击选择/取消选择
- 选中状态清晰可见

### 3️⃣ 任务管理
- 批量创建发布任务
- 支持定时发布
- 实时监控任务状态
- 查看详细执行日志

## 🚀 快速开始

### 启动服务
```bash
# 启动后端
cd server && npm run dev

# 启动前端（新终端）
```

```
http://localhost:5173/publishing-tasks
```

### 使用流程
```
1. 选择文章 → 2. 选择平台 → 3. 创建任务
```

## 💡 使用示例

### 示例1: 发布单篇文章到多个平台
```
1. 勾选1篇文章
2. 点击3个平台卡片
3. 点击"创建发布任务"
4. 系统创建3个任务并自动执行
```

### 示例2: 批量定时发布
```
1. 全选10篇文章
2. 选择2个平台
3. 设置明天上午10:00
4. 创建20个定时任务
5. 系统在指定时间自动执行
```

## ❓ 常见问题

### Q: 页面显示"暂无草稿文章"？
**A**: 请先到"文章管理"或"生成文章"页面创建草稿文章。

### Q: 页面显示"暂无已登录平台"？
**A**: 请先到"平台登录"页面完成平台登录。

### Q: 如何查看任务失败原因？
**A**: 点击任务行的"日志"按钮，查看详细的错误信息。
