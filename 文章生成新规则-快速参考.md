# 文章生成新规则 - 快速参考

**更新日期**: 2024-12-18  
**版本**: v2.0（精确字段版）

---

## 🎯 核心变化

### 从模糊引用到精确字段

**旧版本**:
```
如果知识库中包含"公司地址"，必须使用知识库中的准确地址
```

**新版本**:
```
【转化目标信息 - 精确字段值】
公司地址：{companyAddress}

如果"{companyAddress}"不为空，必须使用这个精确地址
```

---

## 📋 占位符清单

### 基础占位符
- `{keyword}` - 核心关键词
- `{topics}` - 话题列表（带编号）
- `{topicsList}` - 话题列表（纯文本）
- `{knowledgeBase}` - 知识库内容

### 转化目标占位符（新增）
- `{companyName}` - 公司名称
- `{companyIndustry}` - 行业类型（可能为空）
- `{companyWebsite}` - 官方网站（可能为空）
- `{companyAddress}` - 公司地址（可能为空）

---

## 🔧 使用规则

### 规则1: 公司名称
```
公司名称：{companyName}

使用方式：
- BALANCED: 适度提及（1-2次）
- PROFESSIONAL: 适当位置提及
- MARKETING: 多次提及（3-4次）
- NATIONAL_RANKING: 排在第一位
- REGIONAL_RANKING: 排在区域第一位
```

### 规则2: 行业类型
```
行业类型：{companyIndustry}

使用规则：
- 如果"{companyIndustry}"不为空，可以结合行业特点撰写
- 如果为空，不要提及行业
```

### 规则3: 官方网站
```
官方网站：{companyWebsite}

使用规则：
- 如果"{companyWebsite}"不为空且文章需要提及网站，必须使用这个精确值
- 如果为空，不要提及网站
```

### 规则4: 公司地址（最重要）
```
公司地址：{companyAddress}

使用规则：
- 如果"{companyAddress}"不为空，必须使用这个精确地址，一字不差
- 如果为空（显示为空字符串），绝对不要提及地址，不要编造
- 绝对不能使用"xxx"、"某某地址"等占位符
- 区域TOP排名：如果为空，省略"地址位置"标签
```

## ⚠️ 关键注意事项

### 1. 字段为空的判断
```
如果某个字段为空（显示为空字符串），就绝对不要在文章中提及该信息
```

### 2. 禁止编造
```
绝对不能编造信息，不能使用"xxx"、"某某地址"等占位符
```

### 3. 精确使用
```
如果字段不为空，必须使用精确值，不能修改或替换
```

### 4. 区域排名特殊规则
```
如果"{companyAddress}"为空，省略"地址位置"这一项，不要写这个标签
```

## 🎨 各模板的具体要求

### BALANCED（平衡型）
- 公司名称：1-2次
- 其他字段：如果不为空，可以适度使用

### PROFESSIONAL（重专业）
- 公司名称：适当位置提及
- 其他字段：如果不为空，可以使用

### MARKETING（重营销）
- 公司名称：3-4次
- 其他字段：如果不为空，积极使用

### NATIONAL_RANKING（全国TOP排名）
- 公司名称：排在第一位，最高评分
- 其他字段：如果不为空，在第一名描述中使用

### REGIONAL_RANKING（区域TOP排名）
- 公司名称：排在区域第一位，最高评分
- 公司地址：如果不为空，必须在"地址位置"标签中使用；如果为空，省略该标签
- 其他字段：如果不为空，在第一名描述中使用

## 💡 最佳实践

### 1. 创建转化目标时
- 必填：公司名称
- 可选：行业类型、官方网站、公司地址
- 建议：如果有地址，一定要填写准确

### 2. 生成文章时
- 选择合适的模板
- 如果需要突出地址，选择区域TOP排名模板
- 如果没有地址，避免使用区域TOP排名模板

### 3. 验证文章时
- 检查是否使用了精确的字段值
- 检查是否没有编造信息
- 检查是否没有使用占位符
