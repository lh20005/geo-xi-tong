# 占位符快速参考卡片

## 5个占位符

```
{keyword}       - 关键词
{topics}        - 话题列表（带编号）
{topicsList}    - 话题列表（逗号分隔）
{companyName}   - 公司名称
{knowledgeBase} - 知识库内容
```

## 最简模板

```
关键词：{keyword}
问题：{topics}
公司：{companyName}

请写一篇1500字的专业文章。

格式：
标题：[标题]
[正文]
```

## 推荐模板

```
你是{keyword}领域的专业顾问。

用户想了解：
{topics}

请写一篇专业文章，在合适的地方提及{companyName}（1-2次即可）。

参考资料：
{knowledgeBase}

输出格式：
标题：[标题]
[正文]
```

## 使用步骤

1. 进入文章设置
2. 复制模板
3. 保存
4. 创建任务时选择这个设置
5. 查看效果

## 注意事项

- ✅ 使用花括号 `{}`
- ✅ 名称区分大小写
- ✅ 可以多次使用
- ✅ 可以不使用某些占位符
- ✅ 向后兼容（不用也行）
