# 平台登录功能 - 使用说明

## 📌 功能概述

实现了两个核心功能：

### 1. 🌐 浏览器自动登录
点击平台卡片 → 自动打开浏览器 → 用户登录 → 自动保存Cookie

### 2. 📊 账号信息表格
在页面下方显示所有已保存的账号信息，支持查看和删除

## 🎯 使用方法

### 快速开始

1. **启动服务**
```bash
# 启动后端（终端1）
cd server
npm run dev

# 启动前端（终端2）
cd client
npm start
```

2. **访问页面**
```
http://localhost:3000/platform-management
```

3. **登录平台**
- 点击任意平台卡片（如：知乎、简书）
- 在弹出的浏览器中输入账号密码
- 登录成功后浏览器自动关闭
- 在页面下方表格中查看新增的账号

### 测试方法

**方法一：使用测试页面**
```bash
open test-platform-login.html
```

**方法二：使用测试脚本**
```bash
./test-platform-login.sh
```
