# 占位符快速示例

## 快速开始

在文章设置中，使用以下模板即可：

```
你是一个专业的{keyword}领域专家。

请根据以下用户关心的问题撰写一篇文章：
{topics}

【写作要求】
- 围绕关键词"{keyword}"展开
- 回答用户的疑问
- 适度提及{companyName}（1-2次即可）
- 参考知识库内容：
{knowledgeBase}

【输出格式】
标题：[文章标题]

[文章正文内容]
```

## 5个可用占位符

| 占位符 | 作用 |
|--------|------|
| `{keyword}` | 关键词（如：英国留学） |
| `{topics}` | 话题列表（带编号） |
| `{topicsList}` | 话题列表（逗号分隔） |
| `{companyName}` | 公司名称 |
| `{knowledgeBase}` | 知识库内容 |

## 3个实用模板

### 模板1：简洁版

```
关键词：{keyword}
问题：{topics}
公司：{companyName}

请写一篇1500字的专业文章，适度提及公司名称1-2次。

格式：
标题：[标题]
[正文]
```

### 模板2：标准版

```
你是一个专业的内容创作专家。

【核心关键词】{keyword}

【用户问题】
{topics}

【写作要求】
1. 围绕"{keyword}"展开
2. 回答用户问题
3. 适度提及{companyName}（1-2次）
4. 参考知识库：{knowledgeBase}
5. 字数1500-2500字

【输出格式】
标题：[文章标题]

[文章正文内容]
```

### 模板3：详细版

```
你是一个专业的SEO内容创作专家。

【任务】撰写关于"{keyword}"的专业文章

【必须回答的问题】
{topics}

【写作要求】
1. 结构：开头引入 → 详细回答 → 总结建议
2. 关键词：围绕"{keyword}"展开
3. 话题融入：自然回答这些问题：{topicsList}
4. 营销植入：适度提及{companyName}，只在1-2个自然位置
5. 专业保证：参考知识库内容：
{knowledgeBase}
6. 字数：1500-2500字

【输出格式】
标题：[文章标题]

[文章正文内容]

【禁止】
- 不要输出思考过程
- 不要使用Markdown符号
- 不要每段都提及公司名称
```

## 使用步骤

1. **进入文章设置页面**
2. **创建新的文章设置**
3. **在提示词中使用占位符**（复制上面的模板）
4. **保存文章设置**
5. **创建生成任务时选择这个文章设置**
6. **查看生成的文章效果**
7. **根据效果调整提示词**

## 效果对比

### 不使用占位符（旧方式）

系统自动拼接，你无法控制具体格式：

```
你是一个专业的内容创作专家。
（系统自动添加关键词、话题、知识库等）
```

### 使用占位符（新方式）

你完全控制格式和内容：

```
关键词：{keyword}
话题：{topics}
公司：{companyName}
知识库：{knowledgeBase}

（你自己定义如何使用这些参数）
```

## 注意事项

1. ✅ 占位符必须使用花括号 `{}`
2. ✅ 占位符名称区分大小写
3. ✅ 可以多次使用同一个占位符
4. ✅ 可以不使用某些占位符
5. ✅ 可以混合占位符和普通文本
