# 文章设置模板按钮 - 快速参考

## 🚀 快速开始

访问：http://localhost:5173/ → 文章设置 → 新建文章设置

## 📋 三个模板对比

| 特性 | ⚖️ 平衡型 | 🎓 重专业 | 📢 重营销 |
|------|----------|----------|----------|
| **颜色** | 🔵 蓝色 | 🟢 绿色 | 🟠 橙色 |
| **公司名称** | 1-2次 | 不提及 | 3-4次 |
| **字数** | 1500-2500 | 2000-3000 | 1500-2000 |
| **适用场景** | 企业博客 | 技术文章 | 产品推广 |
| **营销程度** | ⭐⭐⭐ | ⭐ | ⭐⭐⭐⭐⭐ |
| **专业程度** | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |

## 🎯 使用步骤

1. 点击"新建文章设置"
2. 点击任意模板按钮
3. 查看自动填充的内容
4. 可选：编辑模板内容
5. 输入设置名称
6. 点击"创建"

## 📝 占位符说明

所有模板都支持以下占位符：

| 占位符 | 说明 | 示例 |
|--------|------|------|
| `{keyword}` | 关键词 | "人工智能" |
| `{topics}` | 话题列表（详细） | "1. 什么是AI\n2. AI的应用" |
| `{topicsList}` | 话题列表（简洁） | "什么是AI、AI的应用" |
| `{companyName}` | 公司名称 | "科技公司" |
| `{knowledgeBase}` | 知识库内容 | "我们提供..." |

## 💡 选择建议

### 写技术教程？
→ 选择 🎓 **重专业**

### 写企业博客？
→ 选择 ⚖️ **平衡型**

### 写产品宣传？
→ 选择 📢 **重营销**

## ⚡ 快捷操作

- **一键应用**：点击按钮即可
- **继续编辑**：应用后可修改
- **重新选择**：可切换其他模板
- **保存使用**：创建后即可用于生成文章

## 🔄 完整工作流

```
创建设置 → 选择模板 → 编辑（可选）→ 保存
    ↓
使用设置 → 选择关键词/话题 → 生成文章
    ↓
占位符替换 → AI生成 → 获得文章
```

## 📊 模板内容预览

### ⚖️ 平衡型
```
你是一个专业的{keyword}领域专家。
【写作要求】
3. 在合适的位置适度提及{companyName}（1-2次即可）
【注意事项】
- 保持专业性，避免过度营销
```

### 🎓 重专业
```
你是一个资深的{keyword}领域专家和内容创作者。
【禁止事项】
- 不要添加营销性质的内容
- 保持纯粹的专业性和客观性
```

### 📢 重营销
```
你是一个专业的营销文案撰写专家。
【品牌植入】
- 在文章中多次自然提及{companyName}（3-4次）
- 在介绍解决方案时重点推荐{companyName}
```

## 🐛 常见问题

**Q: 模板可以修改吗？**  
A: 应用后可以编辑，但原始模板不变，随时可重新调用。

**Q: 占位符会被替换吗？**  
A: 是的，生成文章时系统会自动替换为实际内容。

**Q: 可以不用模板吗？**  
A: 可以，直接输入自定义提示词即可。

**Q: 三个模板可以混合使用吗？**  
A: 可以，应用一个模板后，可以手动编辑融合其他模板的内容。
