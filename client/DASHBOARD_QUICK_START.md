# 工作台页面 - 快速启动指南

## 🚀 立即查看新设计

### 1. 启动前端服务
```bash
cd client
npm run dev
```

### 2. 访问工作台
打开浏览器访问：`http://localhost:5173/dashboard`

## ✨ 新设计亮点

### 专业的顶部导航栏
- 品牌图标 + 页面标题
- 实时更新时间显示
- 时间范围选择器
- 刷新按钮

### 快捷操作区
一键访问常用功能：
- 🔥 新建蒸馏
- 📝 生成文章  
- 🚀 发布任务
- ⚙️ 系统设置

### 重新设计的核心指标卡片
- **关键词蒸馏**：蓝色主题 #1890ff
- **文章生成**：紫色主题 #722ed1
- **发布任务**：青色主题 #13c2c2
- **发布成功率**：绿色主题 #52c41a（带进度条）

每个卡片显示：
- 总数
- 今日新增
- 增长率（带上升/下降图标）

### 优化的图表展示
所有图表采用统一的专业风格：
- 白色卡片背景
- 柔和的阴影效果
- 统一的配色方案
- 清晰的数据标签
- 流畅的悬停效果

## 🎨 设计特点

### 配色方案
```
主色调：商务蓝 #1890ff
辅助色：紫色 #722ed1、青色 #13c2c2
成功色：绿色 #52c41a
警告色：橙色 #faad14
错误色：红色 #ff4d4f
```

### 视觉效果
- 渐变背景：#f5f7fa → #e8ecf1
- 卡片圆角：8px
- 卡片阴影：0 2px 8px rgba(0,0,0,0.08)
- 悬停效果：上浮4px + 阴影加深

### 响应式布局
- 大屏（xl）：4列指标卡片
- 中屏（sm）：2列指标卡片
- 小屏（xs）：1列指标卡片

## 💡 使用提示

1. **时间范围选择**：支持7天、30天、90天和自定义范围
2. **卡片点击**：点击核心指标卡片可跳转到详情页
3. **数据刷新**：点击右上角刷新按钮更新数据
4. **响应式**：支持各种屏幕尺寸，自动调整布局

## 🐛 问题反馈

如遇到问题，请检查：
1. Node.js 版本 >= 16
2. 依赖是否正确安装：`npm install`
3. 后端服务是否正常运行
4. 浏览器控制台是否有错误信息
