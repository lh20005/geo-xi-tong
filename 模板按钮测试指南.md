# 文章设置模板按钮 - 测试指南

## ✅ 功能已完成

文章设置模板按钮功能已成功实现，现在可以进行测试。

## 🎯 功能说明

在"文章设置"页面，点击"新建文章设置"或"编辑文章设置"时，对话框中会显示三个模板按钮：

1. **⚖️ 平衡型** - 专业内容与适度营销相结合（公司名称1-2次）
2. **🎓 重专业** - 纯专业内容，深度分析，不做营销推广
3. **📢 重营销** - 以营销推广为主（公司名称3-4次）

## 🧪 测试步骤

### 1. 访问前端页面
```
http://localhost:5173/
```

### 2. 进入文章设置页面
- 在左侧导航栏找到"文章设置"菜单
- 点击进入文章设置管理页面

### 3. 测试新建功能
1. 点击"新建文章设置"按钮
2. 在弹出的对话框中，查看是否显示三个模板按钮
3. 依次点击每个模板按钮，验证：
   - ✅ 提示词文本框自动填充对应模板内容
   - ✅ 显示成功提示消息
   - ✅ 模板内容完整且格式正确

### 4. 测试编辑功能
1. 选择一个已有的文章设置
2. 点击"编辑"按钮
3. 在编辑对话框中，点击模板按钮
4. 验证模板可以覆盖原有内容

### 5. 测试查看模式
1. 点击"查看"按钮
2. 验证在查看模式下，模板按钮不显示（只读模式）

### 模板内容验证

#### 平衡型模板应包含：
- `{keyword}` 占位符
- `{topics}` 占位符
- `{companyName}` 占位符
- `{knowledgeBase}` 占位符
- 公司名称提及1-2次的说明

#### 重专业模板应包含：
- `{keyword}` 占位符
- `{topics}` 和 `{topicsList}` 占位符
- `{knowledgeBase}` 占位符
- 纯专业内容，无营销要求

#### 重营销模板应包含：
- `{keyword}` 占位符
- `{topics}` 和 `{topicsList}` 占位符
- `{companyName}` 占位符（多次提及）
- `{knowledgeBase}` 占位符
- 营销策略说明

### 用户体验
- [ ] 模板按钮位置合理
- [ ] 提示词文本框足够大（16行）
- [ ] 字体使用等宽字体，便于阅读
- [ ] 可以在应用模板后继续编辑

## 🔍 预期结果

### 平衡型模板示例
```
你是一个专业的{keyword}领域专家。

【用户关心的问题】
{topics}

【写作要求】
1. 围绕关键词"{keyword}"展开，确保内容专业、有价值
2. 详细回答用户的每个问题，提供实用建议
3. 在合适的位置适度提及{companyName}（1-2次即可）
...
```

### 重专业模板示例
```
你是一个资深的{keyword}领域专家和内容创作者。

【核心任务】
撰写一篇关于"{keyword}"的深度专业文章...

【禁止事项】
- 不要添加营销性质的内容
- 保持纯粹的专业性和客观性
```

### 重营销模板示例
```
你是一个专业的营销文案撰写专家...

【品牌植入】
- 在文章中多次自然提及{companyName}（3-4次）
- 在介绍解决方案时重点推荐{companyName}
...
```

## 🐛 常见问题排查

### 问题1：模板按钮不显示
**检查**：
- 确认前端服务正在运行
- 检查浏览器控制台是否有错误
- 刷新页面重试

### 问题2：点击按钮无反应
**检查**：
- 打开浏览器开发者工具
- 查看Console是否有JavaScript错误
- 检查Network请求是否正常

### 问题3：模板内容不完整
**检查**：
- 查看 `client/src/constants/promptTemplates.ts` 文件
- 确认模板常量定义完整
- 重启前端服务
